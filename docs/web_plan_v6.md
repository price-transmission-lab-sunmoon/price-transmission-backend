## 1. 서비스 개요

### 1.1 서비스 정의

분석 파이프라인(Phase 0~9) 결과물을 단계별로 투명하게 시각화하여 제공하는 **가격 전달 이상 탐지 시각화 웹 서비스**다. 계량경제학 분석 수치와 ML 판정 결과를 직접 노출하며, 분석 결과의 해석은 사용자가 직접 수행한다. 계량경제학·ML 분석 과정은 백엔드에서 완결되며, 프론트엔드는 그 결과를 있는 그대로 시각화하는 구현체로 기능한다.

### 1.2 서비스 목적

- **분석 투명성 확보**: 가격 전달 구조(국제가→수입단가→PPI→CPI)의 단계별 이상 탐지 결과와 그 판정 근거(통계 수치, ML 이상 점수)를 직접 노출하여, 분석 파이프라인의 재현 가능성과 검증 가능성을 보장한다.
- **정보 접근성 향상**: 체감 물가와 공식 통계 간 괴리의 원인을 단계별로 직접 확인할 수 있는 정보 기반을 마련한다.
- **파이프라인 시각화 구현체**: 연구 논문에서 제안하는 통계-ML 앙상블 이상 탐지 프레임워크의 결과를 인터랙티브하게 탐색할 수 있는 도구를 제공한다.

> **고지 원칙**: 탐지 결과는 참고 정보로 제공되며, 인과관계의 확정이나 가격 예측을 보장하지 않음을 서비스 내 명시한다.

### 1.3 플랫폼

- **데스크탑 웹** (1차 출시 기준)
- 모바일 반응형: 1차 출시 범위 외

### 1.4 서비스 목표

| 목표 | 역할 |
|------|------|
| 캡스톤 결과물 | 분석 파이프라인 결과물의 전문 시각화 구현체 |
| 학술 논문 | 통계-ML 앙상블 프레임워크의 인터랙티브 검증 도구로서의 설계 근거 |

## 2. 분석 데이터 구조

### 2.1 분석 대상 품목 (10종 고정)

| 유형 | 품목 | 분석 경로 | 구간 수 |
|------|------|-----------|---------|
| 3구간 품목 | 밀, 옥수수, 대두, 팜유, 설탕, 커피, 소고기 | A → B → D′ | 3 |
| 4구간 품목 | 땅콩, 바나나, 오렌지 | A → B → C → D | 4 |

### 2.2 이상 탐지 결과 단위 구조

| 속성 | 설명 |
|------|------|
| 품목 | 10종 중 하나 |
| 구간 | A / B / C / D / D′ |
| 시점 | 연·월 |
| 패턴 | 패턴 1 / 패턴 2 / 패턴 3 |
| 신뢰도 등급 | 고신뢰 / 중신뢰 / 참고 |
| 통계 지표값 | 전이율, Z-score, IQR 판정, ECT, IRF 피크 시차, TECM α⁺/α⁻ 등 |
| ml_vote | ML 3모델 탐지 합산 0~3 |
| ml_detail | 모델별 이상 점수 (Isolation Forest, LOF, One-Class SVM 각각) |
| is_new | 이번 달 갱신에서 신규 탐지 여부 |

### 2.3 데이터 갱신

월별 배치 갱신. 화면 고정 위치에 "2026년 3월 기준 · 다음 갱신 4월 15일 예정" 형태로 기준 시점과 다음 갱신 예정일을 함께 표시한다.

### 2.4 분석 시작 연도

품목별 개별 시작 시점을 사용한다. 각 품목의 분석 시작 시점은 해당 품목의 분석 경로 상 모든 구간 데이터가 동시에 가용한 가장 이른 월로 결정한다. 구체적인 시작 시점은 S1 스프린트 내 Phase 0 완료 후 품목별로 확정한다.

웹 UI의 "전체" 기간 버튼 클릭 시 해당 품목의 실제 시작 시점부터 현재까지 표시한다.

## 4. 메인 시각화

### 4.1 흐름 보기 — 스트림 그래프

#### 역할

품목별 가격 전달 구간(A·B·D′)의 전이율 시계열을 연속 곡선으로 표현한다. 이상 시점에 곡선이 솟아오르고 이상 노드가 표시되어, "언제, 어느 단계에서 이상이 있었는가"를 한눈에 파악할 수 있다.

#### 차트 구성

- **X축**: 시간 (품목별 데이터 가용 시작 시점 ~ 현재, 월별). 초기 표시 범위는 현재 기준 3년.
- **Y축**: 구간별 전이율 (직접값 표시. 품목·구간별 실제 전이율 수치를 그대로 사용)
- **곡선**: 구간 A(청색), 구간 B(녹색), 구간 D′(주황). 구간 on/off 토글에 따라 개별 표시 제어. 곡선 아래 그라디언트 채움
- **보조 품목 오버레이**: 보조 품목 선택 시 동일 차트에 보조 품목 곡선을 투명도 40%로 렌더링. 보조 품목의 이상 노드도 투명도 40%로 함께 표시하며 마우스 선택 가능. 주 품목 색상과 보조 품목 색상을 명확히 구분하기 위해 보조 품목은 별도 색상 팔레트(청록·보라·분홍) 사용
- **이상 노드**: 곡선 위에 위치. 신뢰도별 색상·크기·글로우로 구분
- **NEW 배지**: 이번 달 갱신에서 신규 탐지된 노드에 표시
- **외부 충격 구간**: 사건 필터 선택 항목에 따라 배경 음영 누적 표시
- **진입 애니메이션**: 화면 진입 및 품목 전환 시 곡선이 왼쪽에서 오른쪽으로 흘러 들어오는 애니메이션

### 4.2 전달 구조 — 연결 산점도

#### 역할

특정 구간에서 상류 가격 변화율(X축)과 하류 가격 변화율(Y축)의 관계를 산점도로 표현한다. 각 점은 1개월 관측치이며, 정상적인 전달이 이루어진 달은 대각선 기준선(전이율 = 1.0) 근처에 위치한다. 기준선에서 멀수록 해당 달에 전달이 과대·과소·비대칭으로 이루어진 것이다.

이 뷰는 전이율의 **분포 구조와 방향성**을 파악하기 위한 것으로, 시계열 흐름이 아닌 구조 탐색 목적으로 사용한다.

#### 차트 구성

- **X축**: 상류 가격 변화율 (%), 해당 구간의 상류 소스
- **Y축**: 하류 가격 변화율 (%), 해당 구간의 하류 소스
- **대각선 기준선**: 전이율 = 1.0인 정상 전달 기준선 (파란 점선). 차트 진입 시 말풍선 "이 선에 가까울수록 정상 전달"을 2초간 표시
- **구역 레이블 및 설명**: 4개 구역에 레이블과 1줄 설명 표시
  - 우상단: "과대 전달 — 상류 상승이 하류에 더 크게 전달됨"
  - 좌하단: "과소 전달 — 상류 하락이 하류에 덜 전달됨"
  - 우하단: "역전 — 상류 상승, 하류 하락"
  - 좌상단: "깃털 패턴 — 상류 하락에도 하류 무반응 또는 상승"
- **구간 전환 탭**: 구간 A / B / D′ 선택 탭. 도매 경유 품목은 C·D 탭 추가
- **뷰 진입 안내**: 차트 상단에 접이식 설명 패널 ("이 그래프는 무엇을 보여주나요?"). 기본 펼침 상태로 진입하며 사용자가 접을 수 있음

#### 접이식 설명 패널 내용

```
[전달 구조 뷰란?]
이 그래프는 특정 구간에서 가격이 '얼마나, 어느 방향으로' 전달됐는지를 보여줍니다.

X축: 상류(앞 단계) 가격의 월별 변화율
Y축: 하류(다음 단계) 가격의 월별 변화율
각 점: 1개월 관측치

파란 점선(기준선)에 가까울수록 상류 변화가 그대로 전달된 정상적인 달입니다.
빨간·주황 점은 이상 탐지 시점이며, 클릭하면 분석 수치를 확인할 수 있습니다.

흐름 보기가 '언제 이상이 있었는가'라면,
전달 구조는 '그 이상이 어떤 형태였는가'를 보여줍니다.
```

#### 시점 애니메이션 슬라이더

차트 하단에 시간 슬라이더 배치.

| 기능 | 설명 |
|------|------|
| 슬라이더 이동 | 해당 시점의 관측치 강조 표시 |
| 과거 궤적 | 슬라이더 이동 시 이전 시점까지의 관측치를 선으로 연결 |
| 재생 버튼 | 자동으로 시점이 앞으로 이동하며 궤적 애니메이션 재생 |

### 4.3 원시 시계열 — 가격 단계 그래프

#### 역할

계량경제학 파이프라인 입력 데이터인 전처리 완료 원시 시계열(국제가·수입단가·PPI·도매가·CPI)을 소스별로 직접 표시한다. 이상 탐지 결과가 어떤 가격 데이터에서 출발했는지를 원천 수준에서 확인할 수 있다.

이 뷰는 전이율 파생값이 아닌 **원데이터 자체**를 보여주므로, 흐름 보기와 병행하여 탐색하면 이상 시점 전후의 실제 가격 움직임을 직접 확인할 수 있다.

#### Y축 단위 처리

모든 소스의 Y축은 **2020년 평균 = 100을 기준으로 계산한 지수**로 통일한다. 소스별 절대 단위(원/kg, 달러/톤, 지수 등)가 상이하므로, 기준 시점 정규화를 통해 단일 Y축에서 5개 소스의 상대적 움직임을 직접 비교 가능하게 한다.

> 기준: 각 소스별로 2020년 12개월 평균을 100으로 설정. 이후 시점 값은 해당 기준 대비 백분율 지수로 환산.

#### 레이아웃 구성

원시 시계열 뷰는 6종 레이아웃 중 하나를 선택해 표시한다. 레이아웃은 필터 바의 레이아웃 선택 영역에서 전환한다. 기본값은 **레이아웃 1**.

| 레이아웃 | 표시 소스 | 표시 구간 | 설명 |
|----------|-----------|-----------|------|
| **레이아웃 1** | 국제가·수입단가·PPI·도매가·CPI | 없음 | 원시 시계열 데이터 5종만 출력. 전체 흐름 개관용 |
| **레이아웃 2** | 국제가·수입단가 | 구간 A | 국제가→수입단가 전달 구조. 구간 A 전이율 오버레이 |
| **레이아웃 3** | 수입단가·PPI | 구간 B | 수입단가→PPI 전달 구조. 구간 B 전이율 오버레이 |
| **레이아웃 4** | PPI·도매가 | 구간 C | PPI→도매가 전달 구조. 구간 C 전이율 오버레이. **4구간 품목(땅콩·바나나·오렌지)만 활성화. 3구간 품목 선택 시 비활성화 및 레이아웃 선택지 그레이아웃** |
| **레이아웃 5** | 도매가·CPI | 구간 D 또는 D′ | 도매가→CPI (4구간 품목: 구간 D) 또는 PPI→CPI (3구간 품목: 구간 D′). 품목 유형에 따라 자동 전환 |
| **레이아웃 6** | 국제가·수입단가·PPI·도매가·CPI | 구간 A·B·C·D·D′ (품목에 따라 상이) | 원시 시계열 5종 + 해당 품목의 전 구간 전이율 오버레이 동시 표시 |

#### 레이아웃별 차트 구성 공통 사항

- **X축**: 시간 (월별). 흐름 보기와 동일한 기간 필터 및 휠 줌 연동
- **Y축**: 2020년 평균 = 100 기준 지수. 단일 Y축 사용
- **소스 곡선**: 소스별 고유 색상 (국제가: 보라, 수입단가: 청색, PPI: 녹색, 도매가: 주황, CPI: 빨강)
- **구간 전이율 오버레이**: 레이아웃 2~6에서 해당 구간의 전이율 곡선을 보조 Y축(우측) 없이 동일 지수 축에 표시. 전이율은 별도 색상(회색 계열 점선)으로 구분
- **이상 노드**: 흐름 보기에서 탐지된 이상 시점을 동일한 형식으로 해당 소스 곡선 위에 표시
- **미니맵**: 흐름 보기와 동일한 미니맵 컴포넌트 재사용

#### 소스별 표시 조건

| 소스 | 색상 | 활성화 조건 |
|------|------|-------------|
| 국제가 (2020=100 지수) | 보라 | 전 품목 |
| 수입단가 (2020=100 지수) | 청색 | 전 품목 |
| PPI (2020=100 지수) | 녹색 | 전 품목 |
| 도매가 (2020=100 지수) | 주황 | 4구간 품목(땅콩·바나나·오렌지)만 활성화. 3구간 품목 선택 시 비활성화 및 해당 레이아웃(4·5) 항목 그레이아웃 |
| CPI (2020=100 지수) | 빨강 | 전 품목 |

## 5. 비교 탐색 — 멀티 품목 오버레이

### 5.1 개요

흐름 보기에서 주 품목과 보조 품목 1개를 동시에 표시하는 2품목 비교 모드. 같은 시기에 두 품목에 이상이 동시에 발생했는지 시각적으로 확인할 수 있다.

멀티 오버레이는 최대 2개 품목(주 + 보조)으로 확정한다.

### 5.2 진입 방법

상단 바의 "비교 추가 +" 버튼 클릭 후 드롭다운에서 보조 품목 1개 선택. 보조 품목이 선택된 상태에서는 상단 바에 보조 품목명과 제거 버튼(×)이 함께 표시된다.

### 5.3 표시 방식

| 항목 | 설명 |
|------|------|
| 주 품목 | 기존 색상 및 불투명도 100%로 표시. 이상 노드 완전 불투명 |
| 보조 품목 | 별도 색상 팔레트 사용 (청록·보라·분홍 중 자동 할당). 전체 투명도 40% 적용 |
| 보조 품목 이상 노드 | 투명도 40% 적용. 마우스 호버·클릭 모두 가능. 클릭 시 분석 수치 패널이 보조 품목 기준으로 갱신 |
| 구간 선택 | 비교 모드에서도 구간 on/off 토글이 주 품목과 보조 품목 모두에 동일 적용 |
| 범례 | 차트 하단에 주 품목 / 보조 품목 색상 범례 표시 |
| 패널 구분 | 분석 수치 패널 헤더에 현재 선택 노드가 주 품목인지 보조 품목인지 배지로 표시 |

### 5.4 품목 클러스터 빠른 선택

보조 품목 드롭다운 내 클러스터 단위 버튼.

| 버튼 | 포함 품목 |
|------|-----------|
| 곡물류 | 밀, 옥수수, 대두 |
| 유지·감미료류 | 팜유, 설탕, 커피 |
| 과일·견과류 | 땅콩, 바나나, 오렌지 |

## 6. 분석 수치 패널

이상 노드 클릭 시 우측에 슬라이드인하는 패널. 해당 시점·구간의 계량경제학 분석 수치, ML 판정 결과, 패턴 판정 경로, IRF 차트를 직접 표시한다.

### 6.1 패널 구조

```
┌─────────────────────────────────────┐
│ [품목] [구간] [시점]          [×]   │
│ [신뢰도] [패턴] [주/보조 배지]      │
│ [NEW 배지 — 이번 달 신규]           │
├─────────────────────────────────────┤
│ § 계량경제학 수치            [▾]    │
│   수치 그리드 + 각 항목 클릭 시     │
│   개별 분석 그래프 인라인 표시      │
├─────────────────────────────────────┤
│ § ML 판정                   [▾]    │
│   ml_vote / 모델별 이상 점수 바 차트│
│   + 각 모델 결과맵 (클릭 시 확장)  │
├─────────────────────────────────────┤
│ § 패턴 판정 경로             [▾]    │
│   Step-by-step 판정 흐름           │
├─────────────────────────────────────┤
│ § IRF 차트                  [▾]    │
│   전체 기간 vs 이상 시점 전후       │
└─────────────────────────────────────┘
```

### 6.2 계량경제학 수치 섹션

해당 시점·구간에서 산출된 분석 수치를 그리드로 표시한다. **각 수치 항목은 클릭 가능하며, 클릭 시 해당 지표의 전체 시계열 분석 그래프가 패널 내에 인라인으로 펼쳐진다.** 통합 그래프가 아닌 지표별 개별 그래프이다.

| 항목 | 표시 내용 | 클릭 시 인라인 그래프 |
|------|-----------|----------------------|
| 전이율 | 해당 월 전이율, 롤링 48개월 평균, 정상 범위(Q1~Q3) | 전이율 시계열 + 롤링 평균선 + 정상 범위 밴드. 이상 탐지 시점 수직선 강조 |
| Z-score | 해당 시점 Z-score 값, 임계값(2.0/2.5) 대비 위치 | Z-score 시계열 + 주의(2.0)·경보(2.5) 수평 임계선 |
| IQR 판정 | Q1, Q3, 상·하한값, 해당 값 위치 | 전이율 분포 박스플롯 (롤링 48개월 기준). 해당 시점 위치 마커 |
| IRF 피크 시차 | 정상 전달 시차 N개월 vs 실제 반응 시차 M개월 | IRF 곡선 (§6.5와 동일) |
| ECT 수준 | 장기 균형 이탈 수준값, N개월 연속 방향 | ECT 시계열 + 기준선(0) |
| TECM α⁺/α⁻ | 상승·하락 조정 계수, Wald 검정 p값 | 상승기/하락기 전이율 분포 비교 (겹친 히스토그램) |
| Bai-Perron | 해당 관측이 속한 하위 기간, 구조 변화 시점 | 전체 전이율 시계열에 구조 변화 시점 수직선 표시 |
| 모형 유형 | VAR 또는 VECM (Johansen 검정 결과 기반) | — (그래프 없음) |

그래프 영역은 클릭으로 토글(펼치기/접기)된다. 여러 항목을 동시에 펼칠 수 있으며 각각 독립적으로 동작한다.

### 6.3 ML 판정 섹션

3개 모델의 판정 결과를 개별 표시한다. 각 모델 행은 클릭 시 해당 모델의 **결과맵(feature space projection)**이 인라인으로 확장된다.

#### 요약 행

| 항목 | 표시 내용 |
|------|-----------|
| ml_vote 요약 | "3개 모델 중 N개 탐지" + 탐지한 모델명 배지 |
| Isolation Forest | 이상 점수(anomaly score) 바 차트. 전체 분포 상 퍼센타일 위치 표시 |
| LOF | LOF score 바 차트. 임계값 대비 위치 표시 |
| One-Class SVM | decision function 값 바 차트. 정상 경계(0) 대비 위치 표시 |

#### 결과맵 (각 모델 행 클릭 시 인라인 확장)

각 모델의 결과맵은 해당 모델이 분석에 사용한 피처 공간에서의 이상도 분포를 2차원으로 시각화한 것이다. 투영 축 구성은 계량경제학 파이프라인 출력 확정 후 S4 스프린트 내 재검토한다. 결과맵은 모델별로 독립적이며, 여러 모델을 동시에 펼칠 수 있다.

> **보류 사항 (OI-15)**: ML 결과맵 투영 축(PCA 2D vs 직접 피처 2종 선택)은 파이프라인에서 출력 가능한 데이터 구조가 확정된 후 S4 스프린트 내 확정한다.

| 모델 | 현재 방향 (미확정) | 설명 |
|------|-----------------|------|
| Isolation Forest | 전이율·Z-score 축 기준 2D 산점도. 이상 점수를 색상 강도로 표현 | 전이율과 Z-score 공간에서 해당 관측치의 고립도를 보여줌 |
| LOF | 전이율·ECT 축 기준 2D 산점도. LOF 점수를 색상 강도로 표현 | 국소 밀도 대비 해당 관측치의 이상도를 보여줌 |
| One-Class SVM | 전이율·상류 변화율 축 기준 2D 결정 경계 맵. 정상/이상 영역 구분 | 학습된 정상 경계를 기준으로 해당 관측치의 경계 위치를 보여줌 |

결과맵 하단에 한 줄 설명 제공: "이 그래프는 [모델명]이 [품목] [구간]의 전체 관측치를 분석한 결과를 보여줍니다."

### 6.4 패턴 판정 경로 섹션

각 이상 판정이 어떤 단계를 거쳐 확정됐는지 단계별로 표시한다.

패턴 2 예시:
```
[판정 경로]
Step 1. 전이율 산출: 해당 월 전이율 = 1.43
Step 2. 롤링 Z-score: 2.71 → 경보 기준(2.5) 초과 ✓
Step 3. IQR 판정: Q3 + 1.5×IQR 상한(1.28) 초과 ✓
Step 4. 두 기준 동시 충족 → 통계 경보 확정
Step 5. ML 탐지: Isolation Forest ✓ / LOF ✓ / One-Class SVM ✗
Step 6. ml_vote = 2 → 통계 O + ML 동시 확인 → 고신뢰 이상
```

## 8. 분석 방법론 소개 탭

### 8.1 개요

상단 바의 "방법론" 탭 클릭 시 메인 시각화 화면을 대체하는 전용 정보 화면으로 전환된다. 이 탭은 서비스에서 사용하는 계량경제학 파이프라인, 이상 탐지 패턴, ML 모델, 신뢰도 판정 기준을 설명하는 참조 문서 역할을 한다.

### 8.2 탭 내 섹션 구성

#### 섹션 1 — 분석 파이프라인 개요

가격 전달 체계(국제가→수입단가→PPI→도매가→CPI)와 Phase 0~9 파이프라인 흐름을 플로우 다이어그램으로 표시한다.

```
[Phase 0] 데이터 수집·전처리
    ↓
[Phase 1] 계절 조정 (STL 분해)
    ↓
[Phase 2] 정상성 검정 (ADF + KPSS)
    ↓
[Phase 3] 공적분 검정 (Johansen)
    ↓         ↓
  VECM      VAR
    ↓         ↓
[Phase 4] 모형 추정 + IRF 산출
    ↓
[Phase 5] Granger 인과 검정 (4구간 품목 구간 C만)
    ↓
[Phase 6] 구조 변화 탐지 (Bai-Perron + Chow Test)
    ↓
[Phase 7] 통계 기반 이상 탐지 (주 방법론)
    ↓
[Phase 7-ML] ML 보조 교차검증
    ↓
[Phase 8] 결과 종합 + 신뢰도 등급화
    ↓
[Phase 8.5] 품목 간 동조성 분석
    ↓
[Phase 9] 논문 작성 + 웹 서비스 시각화
```

**플로우 다이어그램 렌더링 방식**: D3.js v7로 인터랙티브 다이어그램 구현을 우선한다. D3.js 구현이 원활하지 않을 경우 Figma에서 제작한 SVG 정적 이미지를 삽입하는 방식으로 대체한다. S6 스프린트 진행 상황에 따라 최종 결정한다.

#### 섹션 2 — 이상 탐지 패턴 3종

각 패턴을 카드 형태로 표시. 패턴 정의, 적용 구간, 판정 기준, 예시를 포함한다.

| 패턴 | 요약 | 적용 구간 |
|------|------|-----------|
| 패턴 1: 방향 역전 및 전달 시차 이탈 | 상류 가격 방향과 하류 가격 방향이 반대이거나, 정상 전달 시차(IRF 피크 시점 + 버퍼 1개월)를 초과해도 하류 무반응인 경우 | 전 구간 (A·B·C·D·D′) |
| 패턴 2: 전이율 이탈 및 비대칭 전달 | 전이율이 롤링 Z-score와 IQR 기준을 동시 초과하거나, TECM/비대칭 VAR에서 상승·하락 조정 속도가 유의미하게 다른 경우 (로켓-깃털 효과) | 구간 A·B |
| 패턴 3: 안정기 스프레드 누적 확대 | 국제가 안정기(원화 환산 월 변동 ±3% 이내)에 수입단가-PPI 간 수준 괴리가 N개월 연속 같은 방향으로 확대되는 경우 | 구간 B |

#### 섹션 3 — 계량경제학 기법 설명

각 기법을 항목별로 설명. 클릭 시 상세 설명 확장.

| 기법 | 한 줄 설명 |
|------|-----------|
| STL 분해 | 계절 성분을 제거해 월별 데이터의 순수한 추세·잔차를 분리하는 전처리 기법 |
| ADF + KPSS 검정 | 시계열이 단위근을 가지는지(비정상인지) 확인. 두 검정을 병행해 결론을 보완 |
| Johansen 공적분 검정 | 두 가격 시계열 간 장기 균형 관계가 존재하는지 확인. 있으면 VECM, 없으면 VAR |
| VAR / VECM | 다변량 시계열 관계를 추정하는 모형. VECM은 장기 균형 복귀 속도를 추가로 측정 |
| IRF (충격반응함수) | 한 변수에 충격을 주었을 때 다른 변수가 얼마나, 얼마나 빨리 반응하는지를 시각화 |
| Bai-Perron | 데이터에서 전이율의 구조적 변화 시점을 자동으로 탐지하는 기법 |
| TECM / 비대칭 VAR | 가격이 오를 때와 내릴 때 전달 속도·크기 차이를 통계적으로 검정 |
| Z-score + IQR 교차 판정 | 롤링 48개월 기준으로 전이율 이탈을 탐지. 두 기준을 동시 충족할 때만 경보 |

#### 섹션 4 — ML 모델 설명

3개 비지도 학습 모델의 역할과 판정 기준을 설명한다.

| 모델 | 작동 원리 | 이 서비스에서의 역할 |
|------|-----------|---------------------|
| Isolation Forest | 피처 공간에서 관측치를 무작위로 분리할 때 적은 분기 횟수로 고립되는 점을 이상으로 판정 | 전이율·변화율 피처 공간에서 극단적으로 이탈한 월 탐지 |
| LOF (Local Outlier Factor) | 주변 이웃의 밀도와 자신의 밀도를 비교해 밀도가 낮은 점을 이상으로 판정 | 구조 변화 이후 하위 기간 내 국소 이상 탐지 |
| One-Class SVM | 정상 관측치가 모인 영역의 경계를 학습하고, 경계 밖 점을 이상으로 판정 | 안정기 피처 분포를 정상으로 학습 후 경계 이탈 시점 탐지 |

**ML 입력 피처 (6종, 전 품목 공통)**: 월별 전이율 · 상류 가격 변화율 · 하류 가격 변화율 · ECT 또는 로그 수준 스프레드 · 환율 월 변동률 · 국제가 월 변동률

ML은 Phase 7의 통계 판정 결과를 입력으로 받지 않는다. 동일한 원시 데이터(Phase 0~4 산출값)를 바탕으로 독립적으로 판정하므로 교차검증 의미가 성립한다.

## 12. 데이터 흐름

```
[공공 데이터 수집]
  World Bank Pink Sheet / FAO FFPI / 관세청 수입단가
  한국은행 ECOS (PPI·CPI) / KAMIS 도매가 / 환율 API
      ↓ 월별 배치 (APScheduler)
[분석 파이프라인 실행]
  Phase 0~9 (Python)
  → 이상 탐지 결과 + 통계 지표값 + ML 이상 점수 + ML 결과맵 데이터
    + IRF 데이터 + 원시 시계열(2020=100 지수 환산 포함) + is_new 플래그 생성
      ↓
[PostgreSQL]
  anomaly_results    — 탐지 결과 + 신뢰도 등급
  stat_metrics       — 구간별 통계 수치 (전이율, Z-score, ECT, IRF 피크 등)
  stat_timeseries    — 지표별 전체 시계열 (전이율·Z-score·ECT 등)
  ml_scores          — 모델별 이상 점수 (IF, LOF, OC-SVM)
  ml_projections     — ML 결과맵 2D 투영 데이터 (모델별·구간별, 축 구성 미확정)
  irf_data           — 구간별 IRF 곡선 데이터 (전체 기간 + 하위 기간별)
  raw_prices         — 원시 시계열 원본값 + 2020=100 지수 환산값 (국제가·수입단가·PPI·도매가·CPI)
      ↓
[FastAPI 백엔드]
  GET  /api/commodities/{id}/stream            → 스트림 그래프 데이터
  GET  /api/commodities/{id}/scatter           → 산점도 데이터
  GET  /api/commodities/{id}/raw-prices        → 원시 시계열 데이터 (레이아웃별 소스·구간 조합)
  GET  /api/anomalies/latest-summary           → 이달의 이상 요약 배너
  GET  /api/anomalies/{id}/stats               → 계량경제학 수치
  GET  /api/anomalies/{id}/stat-series/{metric} → 지표별 개별 시계열
  GET  /api/anomalies/{id}/ml                  → ML 판정 수치
  GET  /api/anomalies/{id}/ml-map/{model}      → ML 결과맵 투영 데이터
  GET  /api/anomalies/{id}/irf                 → IRF 차트 데이터
  GET  /api/anomalies/{id}/path                → 패턴 판정 경로
      ↓
[React 프론트엔드]
  D3.js    → 스트림 그래프 / 산점도 / 원시 시계열 / IRF 차트
             / ML 결과맵 / 지표별 인라인 그래프 / 미니맵 / 줌·브러시
             / 방법론 탭 플로우 다이어그램 (D3.js 우선, 원활하지 않을 시 SVG 이미지 대체)
  Zustand  → 필터·품목·보조품목·뷰·패널·오버레이·레이아웃 상태 관리
  패널     → 슬라이드인, 너비 드래그, 섹션 접기/펼치기, 지표 그래프 토글,
             ML 결과맵 토글
```
