## 1. 개요

Phase 5는 4구간 품목(땅콩·바나나·오렌지) 구간 C(PPI↔도매가)에서 양방향 Granger 인과 검정을 수행하여 분석 방향을 확정하는 단계이다. Phase 6는 전 품목·전 구간의 전이율 시계열에서 구조 변화 시점을 탐지하고, 하위 기간별 VAR/VECM을 재추정하여 기간별 기준선을 산출하는 단계이다.

```
Phase 4 산출물 (baseline, model_routing.json)
Phase 1 산출물 (changes, seasonal_adjusted)
    │
    ▼
[Phase 5] Granger 인과 방향 확정
    ├─ 4구간 품목 구간 C: 양방향 Granger 인과 검정
    ├─ 3구간 품목: PL-P5-001 INFO 로그 후 PHASE_SKIP
    └─ granger_direction.json 생성
    │
    ▼
[Phase 6] 구조 변화 탐지 및 기간 분할
    ├─ 전이율 산출 (안정 구간 필터 적용)
    ├─ Bai-Perron (Dynp + BIC): 데이터 주도 변화 시점 탐지
    ├─ Chow Test (2008·2020·2022): 교차 확인
    ├─ 하위 기간 분할 (MIN_SUBPERIOD_OBS=60 미만 시 병합)
    └─ 하위 기간별 VAR/VECM 재추정 (Phase 4 함수 재사용)
    │
    ▼
Phase 7 (이상 패턴 탐지) 입력
```

## 3. Phase 5 — Granger 인과 방향 확정

### 3.1 목적

Phase 4까지의 파이프라인에서 구간 A(국제가→수입단가), B(수입단가→PPI), D/D'(도매가/PPI→CPI)의 분석 방향은 경제 이론에 의해 상류→하류 단방향으로 확립되어 있다.

- **구간 A**: 소국 개방경제 모형(Small Open Economy Model). 한국은 밀·옥수수·대두 등 주요 곡물의 자급률이 극히 낮아(밀 1%, 옥수수 0.8%) 국제 시장에서 가격 수용자(price taker)로 행동한다. 국제 가격이 일방적으로 수입 가격을 결정하며, 역방향(한국 수입단가 → 국제 가격)은 한국의 수입 물량이 글로벌 거래량 대비 미미하므로 성립 불가하다.
- **구간 B**: 비용 전가 이론(Cost Push Theory). 원재료 가격이 생산 비용의 일부로 투입되어 생산자 가격에 반영되는 메커니즘이다. 수입 밀 가격이 오르면 밀가루 생산 비용이 증가하고 이것이 PPI에 반영된다.
- **구간 D/D'**: 가격 전달 이론(Price Transmission Theory). 생산자·도매 단계의 비용이 최종 소매 가격에 전달되는 비용 전가(cost push) 경로가 지배적이다. 역방향인 수요 견인(demand pull) — 소비자 가격이 생산자 가격을 끌어올리는 경로 — 은 개별 품목 수준에서는 현실적으로 약하다.

그러나 **구간 C(PPI↔도매가)는 인과 방향이 이론적으로 불명확하다.** PPI와 도매가는 유통 체계상 거의 동일한 단계에 위치하여, 두 가지 경로가 모두 가능하다. 도매 시장(가락시장) 경매 가격이 생산자 출하 가격을 선도할 수도 있고(시장 가격 발견 기능), 반대로 생산 비용이 도매 출하 가격을 결정할 수도 있다(비용 전가). 어느 쪽이 선행하는지는 품목·시기에 따라 다를 수 있어서, 실증적 확인이 필요하다.

Phase 5는 구간 C에 대해서만 양방향 Granger 인과 검정을 수행하여, 분석 방향을 데이터로 확정한다. 이 결과는 Phase 7에서 구간 C에 어떤 이상 패턴을 적용할지(패턴 1만 vs 패턴 1·2·3 모두)를 결정하는 데 직접 사용된다.

| --------------------------- | -------------------------------- | --------------------------------- |
| 4구간 (has_wholesale=true)  | 3종 (groundnuts, banana, orange) | 구간 C Granger 검정 수행          |

### 3.3 검정 방법

- 함수: `statsmodels.tsa.stattools.grangercausalitytests`
- 유의수준: 5%
- 최대 시차: `model_routing.json`의 해당 구간 `var_lag_aic`
- 입력: `{cid}_changes.csv`의 `ppi_pct`, `wholesale_price_pct` 컬럼

**Granger 인과 검정의 원리:**

"상대 변수의 과거값을 추가하면 예측이 개선되는가?"를 검정한다. 예를 들어 "PPI가 도매가를 Granger-인과하는가?"를 검정할 때, 두 모형을 비교한다.

```
제한 모형:   도매가_t = a₁×도매가_{t-1} + a₂×도매가_{t-2} + ... + 오차
비제한 모형: 도매가_t = a₁×도매가_{t-1} + a₂×도매가_{t-2} + b₁×PPI_{t-1} + b₂×PPI_{t-2} + ... + 오차
```

PPI의 과거값(b₁, b₂, ...)을 추가했을 때 예측 오차가 유의미하게 줄었는지를 F검정으로 판정한다.

**statsmodels 입력 규약:** `grangercausalitytests(data, maxlag)`에서 `data[:, 0]`이 종속변수(반응), `data[:, 1]`이 독립변수(원인 후보). 귀무가설은 "독립변수가 종속변수를 Granger-인과하지 않는다". 시차 1~max_lag 각각에 대해 F검정을 수행하고, p값이 가장 작은 시차의 결과를 채택한다.

**F통계량과 p값의 의미:**

- **F통계량**: 비제한 모형이 제한 모형보다 예측 오차를 얼마나 줄였는지의 비율. F > 1이면 과거값 추가가 예측을 개선한 것이고, F < 1이면 오히려 악화된 것이다.
- **p값**: 이 정도의 F통계량이 우연히 나올 확률. p < 0.05면 "우연으로 설명하기 어려우니 인과가 있다고 판단"(유의), p ≥ 0.05면 "우연일 수 있으니 인과가 있다고 단정할 수 없다"(비유의).

### 3.4 `confirmed_direction` 판정 로직

| PPI→도매가 | 도매가→PPI | confirmed_direction | 예외 코드 | Phase 7 적용 범위                           |
| ---------- | ---------- | ------------------- | --------- | ------------------------------------------- |
| ✔ 유의     | ✗ 비유의   | `ppi_to_wholesale`  | —         | 패턴 1·2·3 모두                             |
| ✗ 비유의   | ✔ 유의     | `wholesale_to_ppi`  | —         | 패턴 1·2·3 모두                             |
| ✔ 유의     | ✔ 유의     | `bidirectional`     | PL-P5-003 | 패턴 1·2·3 모두 (PPI→도매가 기본 방향 유지) |
| ✗ 비유의   | ✗ 비유의   | `none`              | PL-P5-002 | 패턴 1만                                    |

`none`일 때 패턴 1만 적용하는 이유: 인과 방향이 확정되지 않은 구간에서 전이율(패턴 2)이나 스프레드(패턴 3)를 논하는 건 "어느 쪽이 상류이고 어느 쪽이 하류인지" 전제가 부재한 상태에서의 분석이라 의미가 없기 때문이다. 패턴 1(방향 역전·시차 이탈)은 단순히 두 변수의 부호와 시차만 비교하므로 방향 전제 없이도 적용 가능하다.

### 3.5 검정 결과

| 품목   | 관측치 | max_lag | PPI→도매가 F (p)  | 도매가→PPI F (p)  | 확정 방향         |
| ------ | ------ | ------- | ----------------- | ----------------- | ----------------- |
| 땅콩   | 98     | 3       | 8.7422 (0.0039) ✔ | 7.3939 (0.0011) ✔ | **bidirectional** |
| 바나나 | 313    | 3       | 0.3858 (0.5350) ✗ | 2.0790 (0.1268) ✗ | none              |
| 오렌지 | 313    | 3       | 0.0388 (0.8439) ✗ | 1.9136 (0.1676) ✗ | none              |

### 3.6 결과 해석

**땅콩 — bidirectional (양방향 유의)**

PPI→도매가 방향에서 F=8.74(p=0.004), 도매가→PPI 방향에서 F=7.39(p=0.001). 양방향 모두 유의수준 0.05를 크게 밑돌아서 양방향 Granger 인과가 확인되었다. 견과가공품 PPI의 과거값이 땅콩 도매가의 미래를 예측하는 데 도움이 되고, 동시에 땅콩 도매가의 과거값도 PPI의 미래를 예측하는 데 도움이 된다는 뜻이다.

이전 데이터(84관측치, max_lag=2)에서는 양방향 비유의(`none`)였으나, 14개월 데이터 확장(98관측치)과 max_lag 변경(2→3)으로 결과가 전환되었다. 이는 소표본 검정에서 실제 존재하는 인과 관계를 포착하지 못했을 가능성이 높다. Phase 7에서 구간 C에 패턴 1·2·3 모두 적용 가능해졌다.
이 결과는 phase5/granger_direction.json에 저장되어 Phase 7에서 구간 C의 패턴 적용 범위를 자동 결정하는 데 사용된다.

**바나나 — none (양방향 비유의)**

PPI→도매가 방향이 F=0.39(p=0.535). F통계량이 1보다 작다는 건, PPI의 과거값을 추가한 모형이 추가하지 않은 모형보다 도매가 예측력이 오히려 떨어진다는 뜻이다. PPI가 도매가를 전혀 선행하지 않는다는 강한 증거이다.

도매가→PPI 방향은 F=2.08(p=0.127). F > 1이라 약간의 예측 개선이 있지만, p=0.127로 유의수준 0.05를 넘어 통계적으로 확신할 수 없다. 이전 데이터에서 p=0.41이었던 것이 0.127로 크게 내려온 추세를 보이며, 데이터 추가 축적 시 유의 전환 가능성이 있다.

313관측치(26년)라는 충분한 표본에서도 비유의인 건 데이터 구조적 원인이 크다. 바나나 PPI는 ECOS에서 '과실류'(404Y014) 합산 항목으로, 사과·배·감귤·포도 등 국산 제철 과일이 모두 포함되어 있다. 바나나 도매가 변동이 PPI 전체에 미치는 영향이 희석되어 신호 대 잡음비(SNR)가 너무 낮다.

**오렌지 — none (양방향 비유의)**

PPI→도매가의 F=0.04(p=0.844)는 6개 검정 중 가장 약한 관계이다. 사실상 0에 가까운 F값은 과실류 PPI 변동이 오렌지 도매가 예측에 아무런 정보도 제공하지 않음을 의미한다. 도매가→PPI 방향의 F=1.91(p=0.168)은 바나나(F=2.08, p=0.127)보다 약간 약한데, 오렌지의 도매 거래량이 바나나보다 적어서 PPI에 미치는 가중치가 더 낮기 때문이다.

3개 품목 모두 "도매가→PPI 방향이 PPI→도매가 방향보다 강하다"는 일관된 비대칭이 관찰된다. 이는 가락시장(매일 경매, 실시간 반응)이 PPI(월별 집계)보다 시장 정보를 빠르게 반영하는 도매 시장의 가격 발견(price discovery) 기능과 일치한다.

## 4. Phase 6 — 구조 변화 탐지 및 기간 분할

### 4.1 목적

Phase 4에서 산출한 기준선(정상 전달 시차, 전이탄력성)은 전체 분석 기간(최대 314개월)을 하나의 구조로 전제하고 있다. 그러나 25년간 가격 전달 구조가 일정했다고 보기 어렵다.

예를 들어 밀 D'(PPI→CPI) 구간의 전체 기간 기준선은 피크 9개월, 탄력성 0.823이다. 이 값의 의미는 "밀가루 PPI가 1% 상승하면, 9개월 후 밀가루 CPI가 0.823% 상승한다"는 것인데, 이 관계가 25년 전체에 걸쳐 동일하게 유지되었는지 확인할 방법이 없었다. 실제로는 2000~2013년에 피크가 24개월이었고, 2020년 이후에는 1개월이었을 수 있다. 평균인 9개월은 어느 기간도 대표하지 못한다.

Phase 7에서 이 단일 기준선을 그대로 사용하면 구체적으로 다음과 같은 문제가 발생한다:

- **오탐**: 2000~2013년에 실제 피크가 24개월인데 기준선이 9개월이면, PPI 변동 후 10개월째 CPI 미반응 시 "시차 이탈"로 경보. 실제로는 24개월 기준 정상 범위 내이다.
- **미탐**: 2020년 이후 실제 피크가 1개월인데 기준선이 9개월이면, PPI 변동 후 3개월째 이미 반응이 끝났는데 "아직 9개월 안이니까 정상"으로 판단. 비정상적으로 빠른 반응을 놓친다.

Phase 6는 이 구조 변화를 데이터 주도 방식으로 탐지하고, 하위 기간별로 모형을 재추정하여 기간별 기준선을 산출한다. Phase 7에서는 각 시점이 속하는 기간의 기준선을 적용하여 위의 오탐·미탐 문제를 해소한다.

### 4.2 전이율 산출

Bai-Perron에 입력할 시계열로 전이율(= 하류 변화율 ÷ 상류 변화율)을 사용한다. 전이율은 상류→하류 전달의 크기를 직접 측정하는 지표로, "전이율이 변했다 = 전달 구조가 변했다"라는 해석이 가능하다. 변화율 시계열 쌍(2변량)을 입력으로 사용하는 방식도 검토했으나, 이 경우 "전달 구조 변화"가 아닌 "상류 가격 변동성 변화"(예: 2008년 곡물 가격 급등)를 구조 변화로 포착할 위험이 있어 채택하지 않았다.

**안정 구간 필터**: 상류 변화율의 절대값이 3% 미만인 월은 전이율을 NaN으로 처리한다. 상류 가격이 거의 움직이지 않은 달에 하류가 조금이라도 변하면 전이율이 극단적으로 커지는(예: 0.01% ÷ 2% = 200) 분모 0 문제를 방지하기 위함이다. 3%라는 임계치는 Phase 7 패턴 3의 국제가 안정 구간 정의(±3%)와 동일한 `STABILITY_THRESHOLD`를 사용하여 파이프라인 내 파라미터 일관성을 확보했다. NaN은 forward-fill(직전 유효값으로 채움) 후 남은 선두 NaN은 backward-fill로 처리하고, 극단값은 ±10 범위로 클리핑한다.

### 4.3 Bai-Perron (Dynp + BIC)

Bai-Perron 검정의 핵심 질문은 두 가지이다: "변화점이 몇 개인가?(k)" 그리고 "어디에 있는가?(위치)".

ruptures 라이브러리의 `Dynp`(동적 프로그래밍)를 사용한다. k를 미리 지정하면 그 k개 변화점의 최적 위치를 동적 프로그래밍으로 찾는다. "최적"이란 각 세그먼트 내부의 잔차제곱합(RSS) 총합이 최소인 분할이다. 이를 k=0(변화점 없음)부터 k=5까지 반복하고, BIC가 최소인 k를 채택하면 "변화점 수"와 "위치"가 동시에 데이터에 의해 결정된다.

BIC(Bayesian Information Criterion)는 모형 복잡도와 적합도 사이의 균형을 잡는 기준이다: `BIC = n × ln(RSS/n) + k × ln(n)`. 앞부분(적합도)은 변화점을 많이 넣을수록 작아지고, 뒷부분(벌점)은 변화점을 많이 넣을수록 커진다. 두 항의 합이 최소인 k가 과적합도 아니고 과소적합도 아닌 최적 변화점 수이다. k=0도 후보에 포함되므로 "구조 변화가 없다"는 결론도 데이터가 결정한다.

| 파라미터        | 값                     | 근거                                                       |
| --------------- | ---------------------- | ---------------------------------------------------------- |
| model           | `normal`               | 세그먼트별 평균 변화를 탐지                                |
| min_size        | 60 (MIN_SUBPERIOD_OBS) | 하위 기간 최소 관측치 — VAR/VECM 재추정에 필요한 최소 표본 |
| max_breakpoints | 5                      | 25년 데이터에서 5개 이상의 구조 변화는 과적합 위험         |

Pelt(penalty 기반)와 KernelCPD(커널 기반 비모수)도 검토했으나, Pelt는 penalty 파라미터 설정에 자의성이 있어 논문에서 방어가 어렵고, KernelCPD는 "무엇이 변했는가"(평균 vs 분산 vs 분포 형태)의 해석이 모호하여 채택하지 않았다. Dynp + BIC는 전역 최적해 보장 + 변화점 수 자동 결정 + 학술적 방어력을 모두 갖추고 있으며, 314개월 데이터에서 계산 시간은 1초 미만이다.

### 4.4 Chow Test

Bai-Perron의 교차 확인용으로, 경제적 사건에 대응하는 3개 고정 시점에서 구조 변화를 검정한다. Bai-Perron이 "어디에서 변했는가?"를 데이터가 알아서 찾는 것이라면, Chow Test는 "이 특정 시점에서 변했는가?"를 직접 묻는 방식이다. 두 방법의 결과가 일치하면 구조 변화 탐지의 신뢰도가 높아진다.

| 시점    | 경제적 사건                      |
| ------- | -------------------------------- |
| 2008-01 | 글로벌 금융위기 + 곡물 가격 급등 |
| 2020-01 | COVID-19 팬데믹                  |
| 2022-01 | 러시아-우크라이나 전쟁           |

H0: 두 기간의 회귀 계수(상수 + 시간 추세)가 동일하다. 전이율 시계열을 해당 시점 기준으로 양분한 후, 전체 모형의 RSS와 양분 모형의 RSS 합을 F검정으로 비교한다. 분석 범위 밖이거나 양분 구간의 관측치가 10개 미만이면 검정을 수행하지 않는다 (PL-P6-002).

### 4.5 하위 기간 분할 및 병합

Bai-Perron이 탐지한 변화 시점을 기준으로 하위 기간을 분할한다. `n_obs` < 60(MIN_SUBPERIOD_OBS)인 기간은 직전 기간에 병합하고, `merged_with` 필드에 병합 대상 ID를 기록한다. 모든 하위 기간이 60 미만이면 단일 기간으로 유지한다 (PL-P6-003).

60개월 기준은 VAR/VECM 재추정에 필요한 최소 표본에서 기인한다. 시차 4, 변수 2개인 VAR 모형의 파라미터가 약 10개이므로, 파라미터 당 최소 6개 관측치(60 ÷ 10)를 확보해야 추정이 안정적이다.

### 4.6 하위 기간 재추정

2개 이상 독립 하위 기간이 있을 때만 수행한다. Phase 4의 `estimate_vecm()`/`estimate_var()` 함수를 재사용(`from phase4_model_estimation import ...`)하며, 전체 기간의 `model_routing.json` 모형 유형(VECM/VAR)을 하위 기간에도 동일 적용한다.

하위 기간별로 공적분을 재검정하여 모형을 독립 결정하는 방식도 검토했으나 채택하지 않았다. 60~80개월 하위 기간에서 Johansen 검정을 수행하면 검정력 부족으로 거의 모든 구간이 VAR로 판정되는데, 이는 "공적분이 사라져서"가 아니라 "검정력이 부족해서"일 가능성이 높다. 또한 기간마다 모형 구조가 다르면 "1기 VECM 탄력성 1.129 vs 3기 VAR 탄력성 -0.311"처럼 측정 대상 자체가 달라져 기간 간 비교가 불가능해진다. 동일 모형 구조에서 파라미터만 비교하는 것이 해석의 일관성을 유지한다.

**시차 축소**: 하위 기간이 짧을 경우 `effective_lag = min(original_lag, n_obs // 10 - 1)`로 시차를 자동 축소하여 자유도 부족을 방지한다.

### 4.7 경계 사례 플래그

### 5.2 구간별 패턴

구간에 따라 구조 변화 탐지 비율이 뚜렷하게 다르다.

| 구간                  | 변화점 탐지 | 비율 | 특성                                     |
| --------------------- | ----------- | ---- | ---------------------------------------- |
| A (국제가→수입단가)   | 0 / 10      | 0%   | 국제 시장 가격 수용, 구조 변화 없음      |
| B (수입단가→PPI)      | 1 / 10      | 10%  | 쇠고기만 (BSE 특수 사건)                 |
| C (PPI↔도매가)        | 0 / 3       | 0%   | 과실류 PPI 노이즈로 전이율 자체가 불안정 |
| D/D' (도매가/PPI→CPI) | 4 / 10      | 40%  | 소비자 단계에서 구조 변화 집중           |

**구간 A — 10개 품목 전부 변화점 없음.** 국제 가격에서 수입단가로의 전달 비율이 25년간 일관되었다. 2008년 금융위기나 2022년 우크라이나 전쟁 때도 가격 "수준"이 급변했을 뿐, 국제가→수입단가의 전달 "비율" 자체는 변하지 않았다.

**구간 B — 쇠고기만 2005-01에 변화점.** 2003년 12월 미국 BSE(광우병) 발생 → 2004년 수입 금지 → 2005년 부분 재개라는 특수 사건으로 수입 구조가 근본적으로 전환된 것과 일치한다. 나머지 9개 품목은 구간 B에서 안정적이다.

**구간 C — 3개 품목 전부 변화점 없음.** Chow 2020에서 땅콩(F=49.2)과 바나나(F=8.6)가 유의하지만 Bai-Perron은 변화점을 잡지 않았다. 이는 해당 시점에서 통계적으로 유의한 차이는 존재하나, BIC 기준으로 모형을 분할했을 때 복잡도 증가(파라미터 추가) 대비 적합도 개선이 충분하지 않아 단일 기간 유지가 더 효율적이라는 판단이다.

**구간 D/D' — 4개 품목에서 변화점 탐지.** 소비자 가격 단계에서 구조 변화가 집중되는 패턴이다. 유통 마진(대형마트 vs 편의점 vs 온라인), 소매 경쟁 구조(쿠팡 등장 이후 가격 경쟁 심화), 정부 물가 안정 정책, 소비자 행동 변화(COVID 이후 외식→가정식 전환) 등 통제 불가 요인이 개입하는 구간의 특성을 반영한다.

### 5.3 Chow Test 유의성 집계

| 시점    | 유의 / 검정 가능 | 비율  | 비고                       |
| ------- | ---------------- | ----- | -------------------------- |
| 2008-01 | 15 / 26          | 57.7% | 커피·땅콩은 범위 밖        |
| 2020-01 | 13 / 30          | 43.3% | 커피 2020-01은 관측치 부족 |
| 2022-01 | 12 / 33          | 36.4% | 전 품목 검정 가능          |

**Bai-Perron과의 교차 확인**: Bai-Perron이 변화점을 탐지한 5개 구간 전부에서 Chow Test도 해당 시점 근처에서 유의하게 나와 결과의 일관성이 확인되었다. 예를 들어 밀 D'에서 Bai-Perron이 2020-11을 잡았고, Chow 2020-01도 F=103.4(p≈0)로 강하게 유의하다. 10개월 차이가 나는 건 COVID 충격이 전달 구조에 반영되기까지 수 개월이 걸렸기 때문이다.

반면 Chow는 유의하지만 Bai-Perron은 변화점을 잡지 않은 경우가 다수 존재한다. Chow는 미리 지정한 시점 하나만 검정하므로 "차이가 있는가?"에만 답하는 반면, Bai-Perron은 BIC로 "모형을 쪼개는 비용 대비 설명력 향상이 충분한가?"까지 판단하기 때문에 더 보수적이다. 작은 차이를 근거로 모형을 쪼개면 각 하위 기간의 관측치가 줄어들어 재추정이 불안정해지므로, BIC가 "쪼개지 마라"고 한 구간은 단일 기간을 유지하는 것이 적절하다.

2008년(57.7%)이 가장 높은 건 글로벌 금융위기가 가격 전달 구조에 가장 광범위한 영향을 미쳤기 때문이다. 2022년(36.4%)이 가장 낮은 건 러시아-우크라이나 전쟁의 영향이 밀·옥수수·팜유 등 특정 품목에 집중되어 있고, 쇠고기·바나나·오렌지에는 직접적 영향이 제한적이었기 때문이다.

| 2013~2020 | +0.677     | 24개월    | PPI 1% 상승 → CPI 0.68% 상승. 속도 유지, 크기 40% 감소                     |
| 2020~2026 | **-0.311** | **1개월** | PPI 1% 상승 → 1개월 후 CPI 0.31% **하락**. 부호 역전 + 속도 급변           |

1기→2기는 전달 속도(24개월)가 유지되면서 크기만 줄었다. 대형마트 PB 제품 비중 증가로 제조사 PPI 상승분의 소매 전가가 어려워진 것과 일치한다. 3기의 부호 역전은 COVID 이후 정부 물가 안정 대책(핵심 식품 가격 모니터링 강화)으로 PPI 상승이 CPI에 전가되지 못하고 유통 단계에서 흡수된 것으로 해석할 수 있다. 다만 64개월이라는 짧은 표본에서의 IRF 추정 불안정성도 배제할 수 없으며, 논문에서 두 해석을 모두 제시하되 표본 크기 한계를 명시해야 한다.

**팜유 D' — 전달 지속 강화**

| 기간      | 탄력성    | 피크  | 전달 비율                      |
| --------- | --------- | ----- | ------------------------------ |
| 2000~2010 | 0.168     | 4개월 | PPI 1% → CPI 0.17% (약한 전달) |
| 2010~2020 | 0.615     | 4개월 | PPI 1% → CPI 0.62% (중간 전달) |
| 2020~2026 | **0.741** | 3개월 | PPI 1% → CPI 0.74% (강한 전달) |

탄력성이 0.168 → 0.615 → 0.741로 지속 증가하고 피크도 4→4→3개월로 빨라졌다. 2000~2010년에는 대기업 과점 구조에서 원재료 비용 변동을 부분적으로만 소매 가격에 반영했으나(16.8%만 전가), 소비자 가격 정보 투명성 향상(가격 비교 앱 등장)과 유통 채널 다양화로 원재료 비용 변동이 더 빠르고 크게 소매 가격에 반영되기 시작했다.

**쇠고기 B — BSE 이후 방향 역전**

| 기간      | 탄력성     | 피크  | 해석                                                         |
| --------- | ---------- | ----- | ------------------------------------------------------------ |
| 2000~2004 | +0.178     | 2개월 | 수입단가 1% 상승 → 2개월 후 PPI 0.18% 상승. 정상적 비용 전가 |
| 2005~2026 | **-0.151** | 3개월 | 수입단가 1% 상승 → 3개월 후 PPI 0.15% **하락**. 방향 역전    |

2003년 미국 BSE 이후 수입 원산지가 미국 중심에서 호주·뉴질랜드로 다변화되면서 전달 구조가 근본적으로 변했다. 역방향 탄력성(-0.151)은 World Bank Pink Sheet의 쇠고기 기준이 뉴질랜드산인데 한국이 실제로 수입하는 건 미국·호주산이라는 원산지 불일치에서 기인한다. 뉴질랜드산 가격이 오를 때 미국·호주산 가격은 다른 방향으로 움직일 수 있어 단순 비용 전가가 성립하지 않는다.

**옥수수 D' ⚠ — 경계 사례**

VECM→VAR 모형 전환된 구간으로 경계 플래그가 부착되어 있다. 이전 300개월 기준에서는 VECM으로 재추정되어 탄력성 1.18/1.21이었으나, 최신 데이터에서는 VAR 기준 0.119/0.358으로 완전히 다른 결과가 나왔다. 이는 모형 선택이 불안정한 구간에서의 재추정 결과가 해석에 주의를 요한다는 경계 플래그의 존재 이유를 실증적으로 보여준다.

Phase 0~6의 데이터 변환 단계:

```
[원시 데이터]  raw levels
    │

    │
    ├──▶  Phase 4 (VAR 추정): 변화율 데이터
    ├──▶  Phase 5 (Granger): ppi_pct, wholesale_price_pct
    ├──▶  Phase 6 (전이율 산출): 하류_pct / 상류_pct

    │
    ├──▶  Phase 6 (참조): estimation_period 확인
    └──▶  Phase 7 (판정 기준): 정상 전달 시차, 탄력성
    │
    ▼  Phase 5: Granger 검정
[granger_direction]  구간 C 확정 방향
    │
    └──▶  Phase 7: 구간 C 패턴 적용 범위 결정
    │
    ▼  Phase 6: 구조 변화 탐지
[breakpoints]  변화 시점, 하위 기간

    │
    └──▶  Phase 7: 기간별 기준선 적용
```
