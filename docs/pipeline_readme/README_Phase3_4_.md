# Phase 3~4 — 공적분 검정 및 모형 추정

> **작성일**: 2026-04-24  
> **목적**: Phase 3(Johansen 공적분 검정)과 Phase 4(VAR/VECM 추정 + IRF 기준선 산출)의 목적, 코드 역할, 실행 방법, 결과 해석을 정리

---

## 1. 개요

Phase 3은 구간별 시계열 쌍의 장기 균형 관계(공적분) 존재 여부를 검정하여 VECM 또는 VAR 모형 경로를 결정하는 단계이다. Phase 4는 결정된 경로에 따라 모형을 추정하고, IRF(충격반응함수)에서 정상 전달 시차와 전이탄력성을 산출하는 단계이다.

```
Phase 2 산출물 (integration_orders.json)
Phase 1 산출물 (seasonal_adjusted, changes)
    │
    ▼
[Phase 3] Johansen 공적분 검정
    ├─ 구간별 시계열 쌍에 Trace + Max-Eigen 통계량 검정
    ├─ 공적분 있음 → VECM 경로
    ├─ 공적분 없음 → VAR 경로
    └─ model_routing.json 생성
    │
    ▼
[Phase 4] 모형 추정 및 기준선 산출
    ├─ VECM 구간: 수준 데이터로 VECM 추정, ECT 산출
    ├─ VAR 구간: 변화율 데이터로 VAR 추정, 로그 스프레드 산출
    ├─ IRF 산출 → 피크 시점 = 정상 전달 시차, 피크 크기 = 전이탄력성
    └─ 기준선 테이블 생성 (Phase 7 이상 탐지 기준)
    │
    ▼
Phase 5 (Granger 인과 방향 확정) / Phase 6 (구조 변화 탐지) 입력
```

## 3. Phase 3 — Johansen 공적분 검정

### 3.1 목적

두 가격 시계열(상류, 하류)이 각각 비정상(I(1))이더라도 둘 사이에 장기 균형 관계가 존재하는지 확인한다. 공적분이 있으면 두 가격이 일시적으로 벌어져도 결국 균형으로 돌아온다는 뜻이며, 이 경우 VECM으로 장기 균형 복귀 속도를 포함한 모형을 추정할 수 있다. 공적분이 없으면 VAR로 단기 동적 관계만 추정한다.

### 3.2 검정 방법

**Johansen 검정** (Engle-Granger 대비 장점):

- 종속변수 선택 문제 없음 (시스템 전체를 한꺼번에 검정)
- 공적분 벡터의 개수까지 파악 가능
- 2변수 시스템에서도 더 강건

**검정 통계량 2종 병행**:

- **Trace 통계량**: "공적분이 r개 이하" 귀무가설. 통계량 > 임계값(5%)이면 기각.
- **Max-Eigenvalue 통계량**: "공적분이 정확히 r개" 검정. 통계량 > 임계값이면 기각.

**판정 로직**:

| Trace  | Max-Eigen | 판정            | 근거                             |
| ------ | --------- | --------------- | -------------------------------- |
| 기각   | 기각      | **공적분 확정** | 둘 다 일치                       |
| 기각   | 미기각    | **공적분**      | Trace 우선 (표본 크기에 더 강건) |
| 미기각 | 기각      | **공적분 없음** | 보수적 처리                      |
| 미기각 | 미기각    | **공적분 없음** | 둘 다 일치                       |

**파라미터**:

| 파라미터  | 값                             | 근거                                                                  |
| --------- | ------------------------------ | --------------------------------------------------------------------- |
| det_order | 0                              | 상수항 포함, 추세 미포함. 가격 전달의 장기 균형에 일정 마진 존재 가정 |
| k_ar_diff | VAR 최적 시차(AIC) - 1, 최소 1 | VECM 시차 = VAR 시차 - 1                                              |
| 유의수준  | 5%                             | 임계값 테이블 인덱스 1                                                |

**비고**: Johansen 검정은 ADF와 달리 p값을 반환하지 않는다. 통계량이 임계값을 초과하면 귀무가설(공적분 없음)을 기각한다.

### 3.3 적분 차수 호환성

구간 쌍의 두 시계열이 모두 I(1)이면 검정이 가장 이상적이다. I(2) 시계열이 포함된 구간은 주의 플래그(`integration_flag`)를 부착하고 검정을 진행한다.

플래그가 부착된 구간 (Phase 2에서 확인된 I(2) 2건):

- 커피 구간 A: intl_price_krw=I(2), import_price_usd=I(1)
- 땅콩 구간 B: import_price_usd=I(1), ppi=I(2)
- 땅콩 구간 C: ppi=I(2), wholesale_price=I(1)

### 3.4 검정 결과

33개 구간 쌍 검정 결과:

| 모형                  | 개수 | 비율  |
| --------------------- | ---- | ----- |
| VECM (공적분 있음)    | 20개 | 60.6% |
| VAR (공적분 없음)     | 13개 | 39.4% |
| 적분 차수 주의 플래그 | 3개  | —     |

과반수에서 공적분이 확인된 것은 이 프로젝트에 유리한 결과이다. VECM 구간에서는 ECT를 직접 산출할 수 있어서 패턴 3(스프레드 누적 확대)의 탐지 정밀도가 올라가고, 패턴 2(비대칭 전달)의 TECM 검정도 적용 가능해진다.

### 3.6 구간별 공적분 비율

| 구간                  | VECM | VAR | 공적분 비율 | 해석                                                        |
| --------------------- | ---- | --- | ----------- | ----------------------------------------------------------- |
| A (국제가→수입단가)   | 7개  | 3개 | 70%         | 커피 VECM 전환으로 비율 상승. 곡물+땅콩+커피                |
| B (수입단가→PPI)      | 7개  | 3개 | 70%         | 전 구간 중 가장 높음(동률). 가공·제조 단계 전달 경로가 뚜렷 |
| C (PPI→도매가)        | 2개  | 1개 | 67%         | 바나나·오렌지 공판장 기반 거래가 안정적                     |
| D/D' (도매가/PPI→CPI) | 4개  | 6개 | 40%         | 전 구간 중 가장 낮음. 소매 단계 유통 마진의 완충 효과       |

구간 A와 B의 공적분 비율이 각각 70%로 가장 높다. 특히 구간 A는 이전(60%)에서 커피의 VECM 전환으로 70%로 상승했다. 구간 D/D'가 40%로 가장 낮은 것은 소매 단계의 유통 마진·광고비·포장비·소매업체 가격 전략이 생산자 물가와 소비자 물가 사이의 장기 균형을 약화시키는 구조적 특성을 보여준다.

**품목별 모형 선택 요약**:

| ------ | ------ | ------ | ------ | --------- | ---------------------- |
| 밀     | VECM   | VECM   | —      | VECM      | 전 구간 VECM           |
| 옥수수 | VECM   | VECM   | —      | VAR       | 상류 VECM, 하류 VAR    |
| 대두   | VECM   | VECM   | —      | VAR       | 상류 VECM, 하류 VAR    |
| 팜유   | VECM   | VECM   | —      | VAR       | 상류 VECM, 하류 VAR    |
| 설탕   | VECM   | VECM   | —      | VAR       | 상류 VECM, 하류 VAR    |
| 커피   | VECM⚠️ | VAR    | —      | VAR       | A만 VECM (I(2) 플래그) |
| 쇠고기 | VAR    | VAR    | —      | VAR       | 전 구간 VAR            |
| 땅콩   | VECM   | VAR⚠️  | VAR⚠️  | VECM      | A·D VECM, B·C VAR      |
| 바나나 | VAR    | VECM   | VECM   | VECM      | A만 VAR                |
| 오렌지 | VAR    | VECM   | VECM   | VECM      | A만 VAR                |

(⚠️ = I(2) 포함 구간, 적분 차수 주의 플래그)

5가지 유형으로 분류된다:

**전 구간 VECM (밀)**: 국제가→수입단가→PPI→CPI 전 경로에서 장기 균형이 확인된 유일한 품목이다. 밀가루가 비교적 단순 가공 제품이라 가격 전달 경로가 직접적이고, 314개월의 충분한 표본이 검정력을 뒷받침한다.

**상류 VECM, 하류 VAR (옥수수, 대두, 팜유, 설탕)**: 구간 A·B에서는 장기 균형이 있지만 구간 D'에서는 없다. 옥수수는 이전 데이터(2024-12)에서 전 구간 VECM이었으나, 14개월 확장 후 D'가 VAR로 전환되어 이 유형으로 재분류됐다. 두부(대두)·식용유(팜유)·설탕·스낵과자(옥수수)의 CPI는 원재료 외에 포장·광고·유통 비용이 상당 부분을 차지하므로 PPI와의 장기 균형이 희석되는 것으로 해석된다.

**A만 VECM, I(2) 플래그 (커피)**: 이전에는 전 구간 VAR이었으나 표본이 61→75개월로 늘면서 구간 A에서 공적분이 새로 잡혔다(Trace 12.08→22.31). 다만 intl_price_krw가 I(2)이므로 적분 차수 불일치 플래그가 부착돼 있어 결과 해석에 주의가 필요하다.

**전 구간 VAR (쇠고기)**: 314개월의 충분한 표본에도 불구하고 어떤 구간에서도 장기 균형이 확인되지 않는다. Pink Sheet(뉴질랜드산)과 한국 수입(미국·호주산)의 원산지 불일치가 구조적 원인이다.

**A·D VECM, B·C VAR (땅콩)**: 이전에는 A만 VECM이었으나 표본 확대(85→99개월)로 구간 D에서 공적분이 새로 잡혔다(Trace 14.54→15.55). 다만 15.55 vs 임계값 15.49로 극도로 아슬한 경계 사례다. 구간 B·C는 ppi가 I(2)로 플래그 부착 상태.

**A만 VAR, B·C·D VECM (바나나, 오렌지)**: 국제가→수입단가 관계는 약하지만, 국내 유통 단계(수입단가→PPI→도매가→CPI)에서는 장기 균형이 확인된다. 국제 열대 과일 시장은 산지별 가격 차이가 크고 물류 변동이 커서 장기 균형이 약하지만, 국내에 들어온 이후에는 공판장(가락시장) 기반 도매 거래가 안정적 전달 채널 역할을 한다는 해석이다.

### 3.8 I(2) 플래그 구간의 검정 결과

| 구간                                     | Trace | 임계값 | 결과   | 비고                                 |

| 커피 A (intl_price_krw→import_price_usd) | 22.31 | 15.49  | VECM⚠️ | I(2) 플래그에도 불구하고 공적분 검출 |

커피 구간 A는 I(2) 플래그가 부착됐음에도 Trace=22.31로 임계값을 크게 상회하여 VECM으로 판정됐다. 표본 확대(61→75개월)의 검정력 향상 효과가 적분 차수 불일치의 영향을 압도한 것으로 보인다. 다만 이론적 전제가 엄밀히 충족되지 않으므로 결과 해석에 주의가 필요하다. 땅콩 B·C는 모두 VAR로, ppi의 I(2) 문제가 공적분 검출을 어렵게 한 영향이 있을 수 있다.

### 3.9 시차 선택 결과

| VAR 최적 시차(AIC) | 구간 수 | 비율  |
| ------------------ | ------- | ----- |
| 1                  | 0개     | 0%    |
| 2                  | 8개     | 24.2% |
| 3                  | 13개    | 39.4% |
| 4                  | 12개    | 36.4% |

시차 3~4가 전체의 76%를 차지한다. 월별 데이터에서 국제가 변동이 국내 가격에 반영되기까지 3~4개월이 걸린다는 의미이며, 곡물 가공업체의 선도 구매 관행(3~7개월)과 일치하는 결과이다. 커피 D'만 시차 1로 유일하게 짧은데, 관측치 75개에서 긴 시차를 선택하면 유효 관측치가 크게 줄어들어 추정 안정성이 떨어지기 때문이다.

### 3.10 Phase 4·7 연계 시사점

**패턴 3 적용 범위**: 구간 B만 패턴 3의 대상이며, VECM 7개 구간에서는 ECT 기반으로, VAR 3개 구간(커피·쇠고기·땅콩)에서는 로그 스프레드 기반으로 스프레드 모니터링을 수행한다. VECM 비율이 70%라 대부분 이론적으로 정밀한 ECT 기반 분석이 가능하다.

**패턴 2 TECM 적용 범위**: TECM(비대칭 전달 검정)은 공적분이 있는 구간에서만 적용 가능하다. 구간 A·B에서 VECM인 14개 구간에 TECM을 적용하고, VAR인 6개 구간에는 비대칭 VAR을 적용한다. TECM 적용 비율 70%.

## **쇠고기의 분석 제약**: 전 구간 VAR이므로 ECT 산출이 불가하고, 패턴 3은 로그 스프레드로 대체하며, 패턴 2는 비대칭 VAR로만 검정 가능하다. 원산지 불일치라는 구조적 원인이므로 데이터 확보만으로는 해결이 어려운 한계다. 커피는 이전에 전 구간 VAR이었으나 구간 A가 VECM으로 전환되어 해당 구간에서는 ECT 기반 분석이 가능해졌다.

## 4. Phase 4 — 모형 추정 및 기준선 산출

### 4.1 목적

Phase 3에서 결정된 경로(VECM/VAR)에 따라 모형을 추정하고, IRF에서 정상 전달 시차(Phase 7 패턴 1 기준)와 전이탄력성(패턴 2 참조)을 산출한다. 또한 ECT(VECM) 또는 로그 수준 스프레드(VAR)를 산출하여 Phase 7 패턴 3 및 Phase 7-ML 피처로 제공한다.

### 4.2 모형 추정 분기

**VECM 구간 (공적분 있음, 20개)**:

- 입력: 계절 조정 **수준** 데이터 (`_sa.csv`)
- 시차: `model_routing.json`의 `johansen_lag` 사용
- 산출: ECT(오차수정항) = 공적분 벡터(β) × 수준 데이터
- ECT는 장기 균형으로부터의 이탈을 직접 측정

**VAR 구간 (공적분 없음, 13개)**:

- 입력: 계절 조정 후 **변화율** 데이터 (`_changes.csv`)
- 시차: `model_routing.json`의 `var_lag_aic` 사용
- 산출: 로그 수준 스프레드 = log(`하류_sa`) - log(`상류_sa`)
- ECT 대체 지표로 Phase 7 패턴 3 및 Phase 7-ML에서 사용

### 4.3 IRF (충격반응함수)

상류 가격에 1단위 충격 발생 시 하류 가격의 시간별 반응 경로를 산출한다.

- **산출 기간**: 0~24개월 (`IRF_HORIZON = 24`)
- **피크 시점** = 정상 전달 시차 → Phase 7 패턴 1에서 "피크 시점 + 1개월 초과 시 시차 이탈" 판정 기준
- **피크 크기** = 전이탄력성 → Phase 7 패턴 2에서 전이율 정상 수준 참조값
- **95% 신뢰구간**: stderr 기반 (IRF ± 1.96 × 표준오차)

**피크 산출 방식**: IRF 절대값이 가장 큰 시점을 피크로 선택. 음수 반응(역방향 전달)도 포착하기 위해 절대값 기준 사용.

### 4.8 시차(lag) 분포

| 시차 | VECM | VAR | 합계 | 비율  |
| ---- | ---- | --- | ---- | ----- |
| 1    | 3개  | 2개 | 5개  | 15.2% |

VECM은 시차 2~3에 집중(16/19)되고, VAR은 비교적 고르게 분포한다. VECM의 시차가 VAR보다 작은 경향이 있는 건 `johansen_lag = var_lag_aic - 1` 때문이다.

VAR에서 시차 4가 나온 4개 구간(쇠고기 A·B·D', 바나나 A)은 단기 동적 관계가 복잡하다는 뜻으로, 4개월 전까지의 과거값이 현재 가격에 영향을 미친다는 것이다. 쇠고기가 3개 구간에서 모두 시차 4인 건 축산물 특유의 복잡한 유통 구조(도축·가공·냉동·해동 과정)가 반영된 결과로 볼 수 있다.

### 4.10 결과 종합

Phase 4 결과에서 확인된 가격 전달 구조의 핵심 특징은 세 가지다.

첫째, 구간 A·B의 곡물 5종+커피는 VECM으로 장기 균형이 확인되고 전달 시차가 길다(6~24개월). 이는 선도 계약 기반 수입 구조와 가공 단계의 가격 조정 지연을 반영한다. 커피 구간 A는 I(2) 플래그가 있으므로 해석에 주의가 필요하다.

둘째, 구간 D'(PPI→CPI)는 밀만 VECM(탄력성 0.823)이고 나머지는 VAR로 단기 전달(1~3개월)만 포착된다. 옥수수가 이전 VECM(과대 전달 1.13)에서 VAR(0.127)로 전환되어, 전 구간 VECM이 밀 단독이 되었다.

셋째, 바나나·오렌지의 과실류 합산 PPI 문제가 구간 B·C에서 비직관적 결과를 유발하며, 데이터 확장 후 심화됐다. 오렌지 구간 C(5.175→5.828)와 바나나 구간 C(-1.272→-2.309)는 ECOS 데이터 구조의 한계에서 기인하며, 해당 구간의 Phase 7 탐지 결과 해석 시 반드시 고려해야 한다.

## 8. 주의사항

### IRF 피크 24개월 도달 케이스

VECM 구간 B에서 대부분 피크가 24개월(최대치)에 찍힌다. 이는 충격 반응이 천천히 누적되면서 IRF_HORIZON 내에 수렴하지 않은 경우다. exception_spec의 PL-P4-003에 해당하며, 해당 구간의 정상 전달 시차는 "≥ 24개월"로 해석해야 한다. Phase 7 패턴 1에서 시차 이탈 판정 시 이 점을 고려한다.

### 구간 A 탄력성의 단위 의존성

구간 A(국제가→수입단가)의 VECM 탄력성이 매우 작은 값(0.0003~0.001)으로 나온다. 이는 국제가(원/톤)와 수입단가($/톤)의 단위가 다르기 때문이며, 가격 전달이 약하다는 뜻이 아니다. Phase 7에서는 탄력성 절대 크기보다 피크 시점(시차)과 방향(부호)을 주로 사용한다.

### 음수 탄력성

쇠고기 구간 B(-0.130), 바나나 구간 A(-0.107), 바나나 구간 C(-2.309) 등에서 음수 탄력성이 나타난다. 이는 상류 가격 상승 시 하류 가격이 오히려 떨어지는 역방향 반응을 의미하며, 원산지 불일치, 과실류 합산 PPI 노이즈, 또는 단기적 대체재 전환 효과가 원인일 수 있다. 바나나 구간 C는 이전(-1.272)에서 현재(-2.309)로 역방향 반응이 심화됐다.

### ECT vs 로그 스프레드

ECT는 공적분 벡터에 의해 이론적으로 정의된 균형 이탈 지표이므로 해석이 명확하다. 로그 스프레드는 공적분이 없는 구간의 대체 지표이며, 두 가격의 로그 비율 변화를 추적한다. Phase 7 패턴 3에서 두 지표의 해석 방식이 다르므로 `ect_type` 필드를 확인하여 분기해야 한다.
