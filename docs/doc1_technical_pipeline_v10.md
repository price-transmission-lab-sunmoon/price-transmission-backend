- v6 → v7: PPI 시작 시점 제약 반영(커피 2019년~, 땅콩 2017년~) — 구간 B 실질 탐지 가능 기간 명시 및 연구의 한계 추가. Phase 8.5 관찰 스토리 3개(곡물 동시 충격·열대 작물 동조성·사료→축산물 시차)로 확정. v6 잔류 오류 수정(쌀 표기 제거). 소고기 A-B-D' 구간으로 변경.

## 1. 사용 기술 목록

### 1.1 시계열 전처리

| 기술 | 목적 | 비고 |
| --- | --- | --- |
| STL 분해 (Seasonal-Trend Decomposition using LOESS) | 계절 성분 제거 → 계절 조정 시계열 생성 | VAR/VECM 입력 전 사전 처리로 적용 |
| ADF 검정 (Augmented Dickey-Fuller Test) | 단위근 존재 여부 확인 → 시계열 정상성 판단 | 비정상 시 1차 차분(전월 대비 변화율) 변환 |
| KPSS 검정 | ADF 검정 보완 → 정상성 재확인 | ADF와 결론이 상충할 경우 보수적 판단(비정상 처리) |

### 1.2 장기 균형 관계 검정

| 기술 | 목적 | 비고 |
| --- | --- | --- |
| Johansen 공적분 검정 | 두 가격 시계열 간 장기 균형 관계 존재 여부 확인 | Trace·Max-Eigen 통계량 병행; 결과에 따라 VAR 또는 VECM 자동 분기 |

### 1.3 모형 추정

| 기술 | 목적 | 적용 조건 |
| --- | --- | --- |
| VAR (Vector Autoregression) | 다변량 단기 동적 관계 추정, 전이탄력성·시차 산출 | 공적분 없음 |
| VECM (Vector Error Correction Model) | 장기 균형 복귀 속도 포함한 동적 관계 추정 | 공적분 있음 |
| IRF (Impulse Response Function) | 충격 전달의 시간 프로파일 시각화, 피크 시점·크기 산출 | VAR/VECM 추정 후 자동 산출 |

### 1.4 인과 방향 및 구조 변화

| 기술 | 목적 | 비고 |
| --- | --- | --- |
| Granger 인과 검정 | PPI↔도매가 구간(구간 C)의 통계적 선행 방향 확정 | KAMIS 도매가 존재 품목(땅콩·바나나·오렌지)에만 적용 |
| Bai-Perron 구조 변화 검정 | 데이터 주도 방식으로 전이율 변화 시점 자동 탐지 | 시점 미지정, 데이터가 결정 |
| Chow Test | 사전 지정 시점 기준 구조 변화 확인 | 주요 외부 충격 시점(2008, 2020, 2022)에 적용 |

### 1.5 통계 기반 이상 패턴 탐지 (Phase 7 — 주 방법론)

3가지 이상 패턴을 정의하고 통계적 기준에 따라 탐지한다. 이 결과가 프로젝트의 주 분석 결과가 된다.

| 기술 | 목적 | 탐지 패턴 |
| --- | --- | --- |
| IRF 시차 기반 방향·시차 판정 | 방향 역전 및 정상 시차 이탈 탐지 | 패턴 1 |
| Z-score + IQR 교차 판정 | 롤링 윈도우(48개월) 기반 전이율 이탈 탐지 | 패턴 2 |
| TECM / 비대칭 VAR | 상승·하락 국면별 전달 비대칭성 검정 | 패턴 2 |
| ECT / 로그 수준 스프레드 | 안정 구간 수준 괴리 누적 탐지 | 패턴 3 |

### 1.6 ML 기반 이상 탐지 (Phase 7-ML — 보조 교차검증)

3개 전통 ML 모델이 Phase 4의 원시 피처를 직접 입력받아 독립적으로 이상 여부를 판단한다. 통계 기반 탐지(Phase 7)의 보조 교차검증 수단으로 활용하며, ML 단독 결과는 주 분석 결과로 사용하지 않는다.

ML은 단일 모델 체계를 유지한다. ML 적용 구간은 구간 A·B로 고정되어 있어 도매가 데이터 가용성과 무관하며, 전 품목에 동일한 ML 파이프라인(피처 6종, 3개 모델)을 적용한다.

| 모델 | 탐지 원리 | 이 프로젝트에서의 역할 | 소표본 적합성 |
| --- | --- | --- | --- |
| Isolation Forest | 트리 기반 고립도 측정 | 전이율·시차 반응 피처 공간에서 극단적 이탈 월 탐지 | 적합 (트리 기반, 100개 이상 관측치에서 작동) |
| LOF (Local Outlier Factor) | 국소 밀도 기반 이상도 측정 | 구조 변화 이후 하위 기간 내 국소 이상 탐지 | 적합 (비모수적, 소표본 내성 있음) |
| One-Class SVM | 정상 영역 경계 학습 | 안정기 피처 분포를 정상으로 학습 → 경계 이탈 탐지 | 적합 (SVM은 소표본에서 강건) |

### 1.7 비대칭성 검정

| 기술 | 목적 | 적용 조건 |
| --- | --- | --- |
| Threshold ECM (TECM) | 상승·하락 국면별 조정 속도 차이 검정 (α⁺ ≠ α⁻) | 공적분 있는 구간 — 주 방법론 |
| 비대칭 VAR (더미 교차항) | 상승·하락기 전이 계수 차이 검정 | 공적분 없는 구간 — 대체 방법론 |
| 구간별 평균 전이율 비교 | 상승기 vs 하락기 평균 전이율 단순 비교 | 보조 근거로만 활용 |

## 2. 분석 대상 품목 및 파이프라인 구조

### 2.1 설계 원칙

한국이 수입에 의존하는 주요 식품 원자재는 국제 원자재 시장에서 원물 또는 반가공 상태로 수입된 뒤, 국내 제조·가공 공장에 B2B 직납되는 유통 구조를 가진다. 이러한 품목은 가락시장 등 공판장 기반 도매 거래를 거치지 않으므로 KAMIS 도매가격 데이터가 구조적으로 생성되지 않는다.

본 연구는 이 시장 구조를 반영하여 **구간 A-B-D′(국제가→수입단가→PPI→CPI)를 주 분석 경로**로 설계한다. KAMIS 도매가가 존재하는 소수 품목에 대해서는 구간 C·D(PPI→도매가→CPI)를 추가 적용하여, 도매 단계에서의 가격 전달 특성을 **보조적으로 비교 분석**한다.

| 1 | 밀 | Wheat, US HRW | 1001 | ✖ | A-B-D′ | 3 |
| 2 | 옥수수 | Maize | 1005 | ✖ | A-B-D′ | 3 |

| 4 | 팜유 | Palm oil | 1511 | ✖ | A-B-D′ | 3 |

| 6 | 커피 | Coffee, Arabica | 0901 | ✖ | A-B-D′ | 3 |
| 7 | 소고기 | Beef | 0201·0202 | ✖ | A-B-D′ | 3 |

### 2.3 분석 구간 정의

| 구간 | 상류 | 하류 | 적용 조건 | 비고 |
| --- | --- | --- | --- | --- |
| A | 국제가 (원화 환산) | 수입단가 | 전체 품목 (10종) | — |
| B | 수입단가 | PPI | 전체 품목 (10종) | — |
| D′ | PPI | CPI | KAMIS 도매가 미존재 품목 (6종) | **주 분석 경로**. B2B 직납 유통 구조를 반영 |
| C | PPI | 도매가 | KAMIS 도매가 존재 품목 (3종: 땅콩·바나나·오렌지) | **보조 비교 경로**. Granger 인과 방향 확정 포함 |
| D | 도매가 | CPI | KAMIS 도매가 존재 품목 (3종) | **보조 비교 경로** |

### 2.4 구간 D′ 상세 명세

구간 D′(PPI→CPI)는 한국 수입 식품 원자재의 실제 유통 구조(B2B 직납)를 반영한 주 분석 경로이다.

- 적용 패턴: 패턴 1(방향 역전·시차 이탈)만 적용
- 패턴 2 제외: CPI 구간은 소매 유통 마진·광고비·포장비 등 통제 불가능 변인이 복합 작용하여 전이율 이탈 원인 귀속 불가
- 패턴 3 제외: 구간 B 단일 구간 적용 원칙 유지
- ML 교차검증 제외: 구간 A·B 한정 원칙 유지
- Phase 5(Granger 인과) 생략: PPI→CPI 방향이 경제 이론상 단방향으로 명확
- 모형 추정: Phase 2~4 동일 절차 (ADF+KPSS → Johansen → VAR/VECM → IRF)

### 2.5 품목별 파이프라인 경로

| 품목 유형 | 해당 품목 | 도매가 | 분석 경로 | 구간 수 |
| --- | --- | --- | --- | --- |
| B2B 직납형 수입 원자재 | 밀, 옥수수, 대두, 팜유, 설탕, 커피, 소고기 | ✖ | 국제가 → 수입단가 → PPI → CPI | A, B, D′ (3개) |
| 도매 시장 경유 수입 신선식품 | 땅콩, 바나나, 오렌지 | ✔ | 국제가 → 수입단가 → PPI → 도매가 → CPI | A, B, C, D (4개) |

공통 구간(A·B)에서의 결과는 전 품목 횡단 비교가 가능하다. 하류 구간(D′ 또는 C·D) 결과는 품목 유형별로 개별 보고하며, 도매 단계 유무에 따른 전달 특성 차이를 보조적으로 분석한다.

## 3. 분석 파이프라인

### 3.1 전체 흐름

```
[Phase 0] 데이터 수집·전처리
    │
    ├─ 소스 수집 → 월별 집계
    │     ├─ 도매가 존재 품목(땅콩·바나나·오렌지): 7개 소스 (KAMIS 포함)
    │     └─ 도매가 미존재 품목(밀·옥수수·대두·팜유·설탕·커피·소고기): 6개 소스
    ├─ 국제가 원화 환산 (월평균 환율 적용)
    ├─ 결측치 선형 보간 (결측률 10% 이상 품목 제외)
    ├─ 품목별 PRODUCT_CONFIG 적재 (구간 리스트, 도매가 유무)
    │
    ▼
[Phase 1] 계절 조정
    │
    ├─ 원(raw) 월별 시계열에 STL 분해 적용
    ├─ 계절 성분 제거 → 계절 조정 시계열 생성
    ├─ 계절 조정 시계열에서 전월 대비 변화율 산출
    │
    ▼
[Phase 2] 정상성 검정
    │
    ├─ ADF 검정 + KPSS 검정 병행
    ├─ 비정상 시 1차 차분 후 재검정
    │
    ▼
[Phase 3] 장기 균형 관계 검정
    │
    ├─ Johansen 공적분 검정
    ├─ 공적분 있음 → VECM 경로
    ├─ 공적분 없음 → VAR 경로
    │
    ▼
[Phase 4] 모형 추정 및 기준선 산출
    │
    ├─ VAR 또는 VECM 추정 (시차: Phase 3 model_routing.json 전달값 사용 — VECM: johansen_lag, VAR: var_lag_aic)
    ├─ IRF 산출 → 피크 시점 = 정상 전달 시차, 피크 크기 = 전이탄력성
    ├─ 구간별 기준선 테이블 생성 (정상 전달 시차, 전이탄력성)
    │
    ▼
[Phase 5] 인과 방향 확정  ← 품목별 분기 시작
    │
    ├─ [도매가 존재 품목: 땅콩·바나나·오렌지]
    │     ├─ Granger 양방향 검정 (구간 C: PPI↔도매가)
    │     └─ 유의미한 방향으로 구간 C 분석 방향 결정
    │
    └─ [도매가 미존재 품목: 밀·옥수수·대두·팜유·설탕·커피·소고기]
          └─ Phase 5 생략 (PPI→CPI 방향은 이론상 단방향으로 전제)
    │
    ▼
[Phase 6] 구조 변화 탐지 및 기간 분할
    │
    ├─ Bai-Perron → 데이터 주도 변화 시점 탐지
    ├─ Chow Test → 주요 외부 충격 시점 교차 확인 (2008, 2020, 2022)
    ├─ 전체 기간 기준선 (베이스라인) 유지
    ├─ 하위 기간별 VAR/VECM 재추정 (최소 60개 관측치)
    ├─ 미달 하위 기간은 인접 기간과 병합
    │
    ▼
[Phase 7] 이상 패턴 탐지 — 통계 기반 (주 방법론)
    │
    ├─ 롤링 윈도우(48개월) Z-score + IQR 산출
    ├─ 초기 48개월은 기준 분포 축적 기간 (탐지 미수행)
    ├─ 교차 판정: 두 기준 동시 충족 시만 경보
    │
    ├─ 패턴 1: 가격 전달 방향 역전·시차 이탈 판정
    │     ├─ [도매가 존재 품목]: 구간 A·B·C·D 전체 적용
    │     └─ [도매가 미존재 품목]: 구간 A·B·D′ 적용
    │
    ├─ 패턴 2: 전이율 크기 이탈·비대칭 전달(로켓-깃털 효과) 판정
```

## 4. 품목 간 클러스터 구조

### 4.1 클러스터 정의

Phase 8.5 이상 동조성 분석에서 사용하는 품목 간 관계 구조이다. Phase 7(통계 기반) 및 Phase 7-ML(ML 기반) 산출물 — 이상 시점, 패턴 유형, 신뢰도 등급 — 을 10개 품목 간 횡단 대조하여 공통 충격 시점의 동조 패턴을 관찰한다. 새로운 분석 방법론을 추가하지 않으며, 기존 탐지 결과의 품목 간 비교만 수행한다.

```
[곡물 클러스터]
  밀 ── 옥수수 ── 대두
  (미국 경작지 경쟁, 우크라이나 공통 충격)

[유지·가공 원료 클러스터]
  대두 ──착유──→ 팜유와 경쟁 (식물성 유지 시장)
  팜유

[열대 작물 클러스터]
  설탕 ── 커피 ── 바나나
  (열대 기후·물류 공통 노출)

[축산물·사료 연결]
  소고기
  (옥수수·대두의 사료 경로를 통한 간접 연결 — 시차 관찰 대상)

[독립 품목]
  땅콩 (견과류 시장 독립)
  오렌지 (아열대 과일, 계절별 원산지 변동)
```

### 4.2 Phase 8.5 관찰 케이스(3개 확정)

**[케이스 A] 곡물 동시 충격 — 2022년 우크라이나 사태**
- 관찰 대상 품목: 밀·옥수수·대두 (곡물 클러스터)
- 관찰 구간: 구간 A (국제가→수입단가)
- 핵심 질문: 2022년 2~4월 세 품목이 동일 기간 내 동시에 이상을 보이는가?
- ML 신뢰도 함의: 동일 기간 k≥3 품목 동시 탐지 → 참고 이상의 신뢰성 상승

**[케이스 B] 열대 작물 동조성 — 2021~22년 브라질 서리**
- 관찰 대상 품목: 커피·설탕·바나나 (열대 작물 클러스터)
- 관찰 구간: 구간 A (국제가→수입단가), 구간 B (수입단가→PPI)
- 핵심 질문: 브라질 서리 충격 기간 세 품목이 동시 이상을 보이는가?
- 데이터 제약: 커피 PPI(구간 B)는 2019년 시작 → 롤링 윈도우 48개월 축적 후 실질 탐지는 2023년~. 브라질 서리(2021~22) 시점은 커피 구간 B 탐지 범위 밖. 구간 A는 제약 없음

**[케이스 C] 사료→축산물 시차 — 곡물 충격의 지연 전달**
- 관찰 대상 품목: 옥수수·대두(상류) → 소고기(하류)
- 관찰 구간: 구간 A (국제가→수입단가)
- 핵심 질문: 옥수수·대두 구간 A 이상 발생 후 소고기 구간 A에서 시차를 두고 이상이 뒤따르는가?
- 주의: 탐색적 관찰에 한정. 사료 경로 인과관계 주장은 본 연구 범위 밖

### 4.3 ML 검증에 대한 기여

ML 단독 탐지(참고 이상)의 해석 한계를 보강한다. ML 단독 탐지 시점이 클러스터 내 다수 품목(k≥3)에서 동시에 발생하면, 통계가 놓쳤지만 실제 이상일 가능성이 상승한다. 반대로 단일 품목에서만 발생하면 노이즈 가능성이 높다.

이를 통해 "참고 이상"을 추가 세분화할 수 있다:
- 참고 이상(ML 단독, 단일 품목) → 약한 신호
- 참고 이상(ML 단독, 다수 품목 동시) → 강한 신호, 추가 검토 대상

## 6. 파라미터 기본값

### 6.1 계량경제학 파라미터

| 파라미터 | 기본값 | 근거 |
| --- | --- | --- |
| VAR 시차 탐색 범위 | 1~4 (AIC 기준, Phase 3에서 결정) | Phase 4는 Phase 3 결정값을 전달받아 사용; 재선택 없음 |

| Z-score 주의 임계값 | > 2.0 | 통계적 5% 수준 근사 |
| Z-score 경보 임계값 | > 2.5 | 통계적 1% 수준 근사 |
| IQR 이상치 기준 | Q1 - 1.5×IQR, Q3 + 1.5×IQR | Tukey 표준 기준 |
| 국제가 안정 구간 | 원화 환산 월 변동률 ±3% 이내 | 재현성 확보를 위한 명시적 기준 |

## 7. Phase 7-ML 설계 원칙

### 7.1 순환 논리 방지

Phase 7-ML의 입력 피처는 Phase 4에서 산출한 원시 지표를 사용하며, Phase 7의 통계 판정 결과(Z-score 경보 여부, IQR 판정 여부 등)는 입력에서 제외한다.

이유: 통계 판정값을 ML에 입력하면, ML은 해당 판정 결과를 그대로 재현하는 방향으로 학습할 가능성이 높다. 이 경우 ML이 추가적인 독립 정보를 제공하지 못하며, 두 방법론이 동시에 탐지했다는 사실의 교차검증 의미가 사라진다.

Phase 7-ML 입력 피처 목록 (전 품목 공통, 6종 고정):

| 피처 | 출처 | 설명 |
| --- | --- | --- |
| 월별 전이율 | Phase 4 | 하류 변화율 ÷ 상류 변화율 |
| 상류 가격 변화율 | Phase 1 | 계절 조정 후 전월 대비 변화율 |
| 하류 가격 변화율 | Phase 1 | 계절 조정 후 전월 대비 변화율 |
| ECT 또는 로그 수준 스프레드 | Phase 4 | 공적분 유무에 따라 분기 |
| 환율 월 변동률 | Phase 0 | 외생 변수 |
| 국제가 월 변동률 (원화 환산) | Phase 0 | 외생 변수 |

### 7.2 ML의 역할 범위

ML은 통계 기반 탐지의 보조 교차검증 수단으로 위치한다. 구체적으로 다음 세 가지 역할을 수행한다.

첫째, 통계 탐지 결과의 신뢰도 보강이다. 통계와 ML이 동시에 탐지한 이상 시점은 고신뢰로 분류하여 논문의 주 결과로 보고한다.

둘째, 통계 미탐지 이상의 후보 제시이다. ML이 탐지했으나 통계가 놓친 시점을 참고 이상으로 기록하여, 통계적 기준의 사각지대를 보완한다.

셋째, 방법론적 기여이다. 계량경제학 기반 이상 탐지에 ML 교차검증을 결합한 앙상블 프레임워크를 제안함으로써, 학술 논문 및 창업 경진대회에서의 차별점을 확보한다.

넷째, Phase 8.5를 통한 간접 검증이다. ML 단독 탐지 시점이 품목 간 클러스터에서 동시에 관찰되면, 해당 시점의 이상 신뢰성이 간접적으로 보강된다.

### 7.3 ML 단독 결과의 해석 제한

ML만 단독으로 탐지한 시점은 참고 이상으로 분류하며, 논문 본문의 주 결과 테이블에는 포함하지 않는다. ML 모델은 비지도 학습 기반이며 경제학적 인과 구조를 반영하지 않으므로, 독립적인 이상 판정 근거로 사용하기에는 해석 가능성이 부족하다. 다만, Phase 8.5에서 다수 품목 동시 탐지가 확인된 ML 단독 시점은 별도 논의 대상으로 기록한다.
