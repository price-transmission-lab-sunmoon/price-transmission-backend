## 1. 연구 배경

### 1.1 문제 제기

국제 원자재 가격 변동은 수입단가 → 생산자물가(PPI) → 도매가 → 소비자물가(CPI)로 단계적으로 전달된다. 한국은 식품 원자재의 상당 부분을 수입에 의존하고 있어, 국제 시장의 가격 변동이 국내 물가 변동으로 직결되는 구조적 취약성을 가진다(김종진 외, 2022).

현실의 가격 전달 과정에서는 이론적 예측과 일치하지 않는 다양한 이상 현상이 관찰된다. 대표적인 현상으로 로켓-깃털 효과, 전달 차단, 안정기 스프레드 누적 확대가 있다. 로켓-깃털 효과는 원자재 가격이 오를 때는 소비자 가격이 빠르게 반응하지만 내릴 때는 느리게 반응하는 비대칭적 전달이다. 전달 차단은 특정 구간에서 가격 변화가 다음 단계로 넘어가지 않는 현상이고, 안정기 스프레드 누적 확대는 국제 원자재 가격이 안정된 상황에서도 국내 가격이 지속 상승하는 현상이다.

기존 국내 연구는 단일 구간 분석 또는 단일 방법론에 국한되는 경우가 많았으며, 다단계 전달 체계 전반을 하나의 파이프라인으로 연결하여 구간별 이탈 패턴을 탐지하는 이상 탐지 프레임워크는 충분히 마련되어 있지 않다.

### 1.2 연구의 필요성

한국은 식품 원자재의 상당 부분을 수입에 의존하고 있어 국제 가격 변동에 대한 노출도가 높다. 공식 소비자물가지수(CPI)는 자가 주거비 미반영·공공요금 억제 등 구조적 괴리 요인으로 인해 실제 물가상승 압력을 과소 반영할 가능성이 있으며(서재용·장용성, 2022), CPI 단일 수치만으로는 가격 전달 이상의 기준점으로 활용하기 어렵다는 제약이 있다.

소비자 입장에서 "왜 올랐는가"를 단계별로 확인할 수 있는 정보 서비스는 충분히 갖춰져 있지 않으며, 가격 형성 정보는 생산자·유통사·정부 등 공급 측에 편중된 구조적 정보 비대칭이 존재한다.

## 2. 연구 목적 및 목표

### 2.1 연구 목적

#### 2.1.1 기술적 목적

국제 원자재가→수입단가→PPI→CPI로 이어지는 가격 전달 체계를 하나의 일관된 분석 파이프라인으로 연결하고, 각 단계 간 전달 과정에서 발생하는 3종 이상 패턴을 정량적으로 자동 탐지하는 시스템을 개발한다.

- 패턴 1: 가격 전달 방향 역전 및 전달 시차 이탈 탐지
- 패턴 2: 전이율 크기 이탈 및 비대칭 전달(로켓-깃털 효과) 탐지
- 패턴 3: 국제가격 안정기 중 하류 물가 스프레드 누적 확대 탐지

분석 방법론: 계량경제학 모형(VAR/VECM, IRF, Johansen 공적분 검정, TECM, Bai-Perron 구조 변화 탐지)을 주 방법론으로, 비지도 ML(Isolation Forest, LOF, One-Class SVM)을 보조 교차검증 수단으로 활용한다.

#### 2.1.2 학술적 목적

계량경제학 기반 통계 탐지와 비지도 ML의 역할을 명확히 분리하고, 두 방법론 간 탐지 결과 수렴 정도에 따라 신뢰도 3단계 등급화 프레임워크를 제안한다.

- 고신뢰 이상: 통계 탐지 + ML 탐지 동시 확인
- 중신뢰 이상: 통계 탐지 확인, ML 미탐지
- 참고 이상: ML 탐지 확인, 통계 미탐지

Ground truth가 존재하지 않는 거시경제 이상 탐지 문제에서 다중 방법론 수렴을 간접 검증 근거로 활용하는 방법론적 기여를 학술 논문으로 발표한다. 추가로, 품목 간 이상 동조성 분석을 통해 ML 단독 탐지의 신뢰성을 간접 보강하는 프레임워크를 제안한다.

#### 2.1.3 사업화 목적

분석 파이프라인 결과를 시각화하고 이상 탐지 근거를 투명하게 제공하는 웹 서비스 원형(Prototype)을 개발한다. 계량경제학 분석 수치와 ML 판정 결과를 단계별로 직접 확인할 수 있는 전문 시각화 도구를 지향하며, 분석 결과의 해석은 사용자가 직접 수행한다.

### 2.2 세부 연구 목표

1. 공공 데이터 소스를 연계하여 가격 전달 체계 데이터셋 구축 (품목별 도매가 가용성에 따라 소스 수 상이)
2. 단계 간 인과 관계 및 전달 시차 정량화
3. 3가지 유형의 이상 패턴 탐지 알고리즘 설계 및 적용
4. 탐지된 이상 패턴의 통계적 유의성 검증 및 ML 교차검증을 통한 신뢰도 등급화
5. 구조 변화 시점 탐지 및 하위 기간별 비교 분석
6. 품목 간 이상 동조성 분석을 통한 공통 충격 패턴 및 ML 탐지 신뢰성 보강
7. 분석 결과를 제공하는 웹 서비스 원형 개발

## 3. 가격 전달 체계 구조

### 3.1 주 분석 경로 — A-B-D′ (3구간)

한국이 수입에 의존하는 주요 식품 원자재 대부분은 국내 제조·가공 공장에 직납되는 유통 구조를 가진다. 이러한 품목에서는 공판장 기반 도매 거래를 거치지 않으며, 실제 가격 전달 경로는 국제가→수입단가→PPI→CPI의 3구간 구조이다. 본 연구는 이 시장 현실을 반영하여 A-B-D′를 주 분석 경로로 설계한다.

```
[국제 원자재 가격 (달러)]
    ↓  (월평균 환율 적용)  ── 구간 A
[수입 단가]
    ↓  (국내 제조·가공)    ── 구간 B
[생산자 물가 (PPI)]
    ↓  (직납 → 소매)      ── 구간 D′
[소비자 물가 (CPI)]
```

### 3.2 보조 비교 경로 — A-B-C-D (4구간)

KAMIS 도매가가 제공되는 품목(땅콩·바나나·오렌지)에 대해서는 도매 단계를 포함한 4구간 경로를 추가 적용하여, 도매 단계에서의 가격 전달 특성을 보조적으로 비교 분석한다.

```
[국제 원자재 가격 (달러)]
    ↓  (월평균 환율 적용)  ── 구간 A
[수입 단가]
    ↓  (국내 제조·가공)    ── 구간 B
[생산자 물가 (PPI)]
    ↓  (도매 유통)         ── 구간 C
[도매가]
    ↓  (소매 유통)         ── 구간 D
[소비자 물가 (CPI)]
```

### 3.3 분석 구간 정의

| 구간 | 상류 | 하류 | 적용 패턴 | 적용 조건 |
|------|------|------|-----------|-----------|
| A | 국제가 (원화 환산) | 수입단가 | 패턴 1, 패턴 2 | 전체 품목 (10종) |
| B | 수입단가 | PPI | 패턴 1, 패턴 2, 패턴 3 | 전체 품목 (10종) |
| D′ | PPI | CPI | 패턴 1 | 3구간 품목 (7종) — **주 분석 경로** |
| C | PPI | 도매가 | 패턴 1 | 4구간 품목 (3종: 땅콩·바나나·오렌지) — **보조 비교** |
| D | 도매가 | CPI | 패턴 1 | 4구간 품목 (3종) — **보조 비교** |

### 3.4 품목별 분석 경로

| 품목 유형 | 해당 품목 | 분석 경로 | 구간 수 |
|-----------|----------|-----------|--------|
| 3구간 품목 | 밀, 옥수수, 대두, 팜유, 설탕, 커피, 소고기 | 국제가 → 수입단가 → PPI → CPI | A, B, D′ (3개) |
| 4구간 품목 | 땅콩, 바나나, 오렌지 | 국제가 → 수입단가 → PPI → 도매가 → CPI | A, B, C, D (4개) |

공통 구간(A·B)에서의 결과는 전 품목 횡단 비교가 가능하다. 하류 구간(D′ 또는 C·D) 결과는 품목 유형별로 개별 보고하며, 도매 단계 유무에 따른 전달 특성 차이를 보조적으로 분석한다.

### 4.2 품목 선정 기준

모든 품목은 다음 3개 필수 조건을 충족한다:

1. **Pink Sheet 국제가 존재**: World Bank Pink Sheet에서 월별 시계열이 제공되어 구간 A(국제가→수입단가)가 성립

## 6. 탐지 대상 패턴 정의

이상 패턴은 3가지 유형으로 정의한다. 각 패턴의 상세 정의 및 판정 기준은 별도 문서(이상 패턴 정의서)에 기술하며, 여기서는 개요만 서술한다.

### 패턴 1 — 가격 전달 방향 역전 및 전달 시차 이탈 탐지

국제 가격의 변화 방향과 국내 가격의 변화 방향이 반대로 나타나거나, IRF에서 산출한 정상 전달 시차에 버퍼 1개월을 더한 기간이 경과해도 하류 가격이 무반응인 경우.

적용 구간: 3구간 품목은 구간 A·B·D′, 4구간 품목은 구간 A·B·C·D 전체.

### 패턴 2 — 전이율 크기 이탈 및 비대칭 전달(로켓-깃털 효과) 탐지

특정 단계에서 전이율이 역사적 정상 범위를 이탈하거나, 상승 시와 하락 시 전달 크기가 체계적으로 불균형한 경우.

구간 A·B에 한정 적용한다. 구간 C·D·D′는 소매 유통 마진 등 통제 불가능 변인이 복합 작용하여 전이율 이탈 원인 귀속이 어려우므로 제외하며, 이를 연구의 명시적 한계로 설정한다.

### 패턴 3 — 국제가격 안정기 중 하류 물가 스프레드 누적 확대 탐지

국제 원자재 가격(원화 환산 월 변동률 ±3% 이내)이 안정적인 시기에 수입단가와 PPI 간 수준 괴리가 N개월 연속 같은 방향으로 확대되는 경우.

구간 B(수입단가→PPI)에만 적용한다.

## 7. 분석 방법론

### 7.1 방법론 개요

| 방법론 | 목적 | 적용 단계 |
|--------|------|-----------|
| STL 분해 | 계절 성분 제거 → 계절 조정 시계열 생성 | Phase 1 |
| ADF + KPSS 검정 | 시계열 정상성 확인 | Phase 2 |
| Johansen 공적분 검정 | 장기 균형 관계 확인 → 모형 선택 분기 | Phase 3 |
| VAR / VECM | 전이탄력성·정상 시차 추정 | Phase 4 |
| IRF (충격반응함수) | 충격 전달 시간 프로파일 산출, 피크 시점 = 정상 시차 | Phase 4 |
| Granger 인과 검정 | PPI↔도매가 방향 확정 (구간 C, 땅콩·바나나·오렌지만) | Phase 5 |
| Bai-Perron / Chow Test | 구조 변화 시점 탐지, 기간 분할 | Phase 6 |
| TECM / 비대칭 VAR | 상승·하락 비대칭성 검정 (패턴 2, 구간 A·B) | Phase 7 |
| Z-score + IQR | 롤링 윈도우 기반 이상치 탐지 (패턴 2) | Phase 7 |
| ECT / 로그 수준 스프레드 | 안정 구간 수준 괴리 모니터링 (패턴 3) | Phase 7 |
| Isolation Forest / LOF / One-Class SVM | ML 기반 보조 교차검증 (구간 A·B, 전 품목 공통) | Phase 7-ML |
| 이상 동조 히트맵 | 품목 간 이상 시점 동시 발생 패턴 시각화 | Phase 8.5 |

### 7.2 분석 절차

```
Phase 0. 데이터 수집 및 전처리  [완료 — 예병성 주도]
    ├─ 소스 수집 → 월별 집계
    │     ├─ 3구간 품목(7종): 6개 소스
    │     └─ 4구간 품목(3종): 7개 소스 (KAMIS 포함)
    ├─ 국제가 원화 환산 (월평균 환율)
    ├─ 결측치 선형 보간 (결측률 10% 이상 품목 제외)
    ├─ 품목별 PRODUCT_CONFIG 적재 (구간 리스트, 도매가 유무)
    ▼
Phase 1. 계절 조정  [개발 중 — 예병성 주도]
    ├─ STL 분해 → 계절 조정 시계열 생성
    ├─ 계절 조정 시계열에서 전월 대비 변화율 산출
    ▼
Phase 2. 정상성 검정
    ├─ ADF + KPSS 병행
    ├─ 비정상 시 1차 차분 후 재검정
    ▼
Phase 3. 공적분 검정
    ├─ Johansen 검정 → 공적분 유무 판정
    ├─ 공적분 있음 → VECM 경로
    ├─ 공적분 없음 → VAR 경로
    ▼
Phase 4. 모형 추정 및 기준선 산출
    ├─ VAR 또는 VECM 추정 (시차: Phase 3 결정값 사용 — VECM: johansen_lag, VAR: var_lag_aic)
    ├─ IRF 산출 → 피크 시점 = 정상 전달 시차
    ├─ 구간별 기준선 테이블 생성
    ▼
Phase 5. 인과 방향 확정  ← 품목별 분기
    ├─ [4구간 품목: 땅콩·바나나·오렌지]
    │     └─ Granger 양방향 검정 (구간 C 방향 확정)
    └─ [3구간 품목: 밀·옥수수·대두·팜유·설탕·커피·소고기]
          └─ Phase 5 생략
    ▼
Phase 6. 구조 변화 탐지 및 기간 분할
    ├─ Bai-Perron + Chow Test
    ├─ 하위 기간별 재추정 (최소 60개 관측치)
    ▼
Phase 7. 이상 패턴 탐지 — 통계 기반 (주 방법론)
    ├─ 패턴 1: 방향 역전·시차 이탈 (전 구간)
    ├─ 패턴 2: 전이율 이탈 + 비대칭 (구간 A·B)
    ├─ 패턴 3: 안정기 스프레드 누적 확대 (구간 B)
    ▼
Phase 7-ML. 이상 탐지 — ML 보조 교차검증 (전 품목 공통)
    ├─ 입력 피처: Phase 0~4 산출값 6종 (구간 A·B)
    ├─ Isolation Forest / LOF / One-Class SVM 독립 실행
    ├─ 통계-ML 교차 대조 및 신뢰도 등급화 (3단계)
    ▼
Phase 8. 결과 종합 및 로버스트니스 체크
    ├─ 신뢰도 등급별 이상 시점 목록 확정
    ├─ 품목 간 횡단 비교 (공통 구간 A·B)
    ├─ 도매 단계 유무에 따른 전달 특성 차이 비교 분석
    ├─ 로버스트니스 체크 (W=36/48/60, STL vs 계절 더미, ML 파라미터)
    ├─ 외부 충격 시점 대조
    ▼
Phase 8.5. 품목 간 이상 동조성 분석
    ├─ Phase 7·7-ML 산출물(이상 시점, 패턴 유형, 신뢰도 등급)을 10개 품목 간 횡단 대조
    ├─ 이상 동조 히트맵 생성 (클러스터별 그룹핑)
    ├─ 관찰 케이스 3개 수행
    │     ├─ [케이스 A] 2022 우크라이나: 밀·옥수수·대두 구간 A 동시 이상 여부
    │     ├─ [케이스 B] 2021~22 브라질 서리: 커피·설탕·바나나 동시 이상 여부 (커피 구간 B 제약 명시)
```

## 8. ML 보조 교차검증 설계

### 8.1 위치 및 역할

ML 기반 이상 탐지(Phase 7-ML)는 통계 기반 탐지(Phase 7)의 보조 교차검증 수단이다. ML 단독 결과는 주 분석 결과로 사용하지 않는다. ML의 역할은 다음 네 가지다.

첫째, 통계 탐지 결과의 신뢰도 보강이다. 통계와 ML이 동시에 탐지한 이상 시점은 고신뢰로 분류한다.

둘째, 통계 미탐지 이상의 후보 제시이다. ML이 탐지했으나 통계가 놓친 시점을 참고 이상으로 기록한다.

셋째, 방법론적 기여이다. 계량경제학 기반 이상 탐지에 ML 교차검증을 결합한 앙상블 프레임워크를 제안한다.

넷째, Phase 8.5를 통한 간접 검증이다. ML 단독 탐지 시점이 품목 간 클러스터에서 동시에 관찰되면, 해당 시점의 이상 신뢰성이 간접적으로 보강된다.

### 8.2 순환 논리 방지

ML 입력 피처는 Phase 0~4에서 산출한 원시 지표만 사용한다. Phase 7의 통계 판정 결과는 입력에서 제외한다.

ML 입력 피처 목록 (전 품목 공통, 6종 고정):

| 피처 | 출처 | 설명 |
|------|------|------|
| 월별 전이율 | Phase 4 | 하류 변화율 ÷ 상류 변화율 |
| 상류 가격 변화율 | Phase 1 | 계절 조정 후 전월 대비 변화율 |
| 하류 가격 변화율 | Phase 1 | 계절 조정 후 전월 대비 변화율 |
| ECT 또는 로그 수준 스프레드 | Phase 4 | 공적분 유무에 따라 분기 |
| 환율 월 변동률 | Phase 0 | 외생 변수 |
| 국제가 월 변동률 (원화 환산) | Phase 0 | 외생 변수 |

### 8.3 ML 단독 결과의 해석 제한

ML만 단독으로 탐지한 시점은 참고 이상으로 분류하며, 논문 본문의 주 결과 테이블에는 포함하지 않는다. 다만, Phase 8.5에서 다수 품목(k≥3) 동시 탐지가 확인된 ML 단독 시점은 별도 논의 대상으로 기록한다.

## 10. 웹 서비스 개요

### 10.1 서비스 정의

분석 파이프라인(Phase 0~9) 결과물을 단계별로 투명하게 시각화하여 제공하는 **가격 전달 이상 탐지 시각화 웹 서비스**다. 계량경제학 분석 수치와 ML 판정 결과를 직접 확인할 수 있도록 하며, 분석 결과의 해석은 사용자가 직접 수행한다.

### 10.2 핵심 기능

- **스트림 그래프**: 품목별 구간(A·B·D′)의 전이율 시계열을 연속 곡선으로 표현. 이상 시점에 신뢰도별 노드 표시
- **연결 산점도**: 상류·하류 가격 변화율 관계를 산점도로 표현. 과대·과소 전달, 비대칭 패턴 직관적 시각화
- **분석 수치 패널**: 이상 노드 클릭 시 해당 시점의 계량경제학 수치(IRF 시차, 전이율, Z-score, ECT 등)와 ML 판정 근거(모델별 이상 점수, ml_vote)를 직접 표시
- **IRF 시각화**: 구간별 충격반응함수 곡선 직접 노출. 정상 기간 IRF와 이상 시점 전후 IRF 오버레이 비교
- **패턴별 판정 경로**: 각 이상 판정이 어떤 단계를 거쳐 확정됐는지 단계별로 표시
- **필터**: 기간·신뢰도·패턴·외부 충격 사건 기준 탐색
- **멀티 품목 오버레이**: 2~3개 품목 동시 비교

## 11. 연구의 한계

### 11.1 분석 품목의 제한

도매가격 데이터가 제공되는 품목은 KAMIS 데이터 구조상 땅콩·바나나·오렌지(3종)에 한정된다. 도매 단계에서의 가격 전달 분석은 이 3종 품목에서만 수행되며, 나머지 7종 품목에 대해서는 도매 단계를 거치지 않는 유통 구조를 반영하여 구간 D′를 적용한다.

커피(원두) PPI는 2019년 1월~, 견과가공품(땅콩) PPI는 2017년 1월~부터 데이터가 제공된다. 롤링 윈도우 48개월 축적 기간을 적용하면 커피 구간 B의 실질 이상 탐지 가능 시점은 2023년 1월~, 땅콩 구간 B는 2021년 1월~로 제한된다. 이로 인해 커피의 경우 2021~22년 브라질 서리 충격 시점이 구간 B 탐지 범위 밖에 위치하며, 구간 A 분석에는 영향이 없다.

바나나·오렌지의 PPI는 개별 품목이 아닌 과실류 합산 항목으로 제공되어, 구간 B에서 다른 과일의 가격 변동이 노이즈로 유입될 수 있다.

### 11.2 ML 방법론의 한계

ML 모델(Isolation Forest, LOF, One-Class SVM)은 비지도 학습 기반으로 경제학적 인과 구조를 반영하지 않는다. ML 탐지 결과는 통계 탐지 결과의 신뢰도 보강 수단으로 활용하며, ML 단독 탐지 결과를 주 분석 결과로 사용하지 않는다.

Phase 8.5의 이상 동조성 분석은 품목 간 이상 시점의 동시 발생 패턴을 탐색적으로 관찰하는 데 한정되며, 품목 간 가격 전이의 인과 분석은 수행하지 않는다. "다수 품목 동시 탐지 = 신뢰성 상승"이라는 논리는 탐색적 수준에서 활용하며, 정량적 검정은 향후 연구로 제안한다.
